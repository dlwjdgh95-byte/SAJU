# 사주 전문가 스킬

자평진전(子平眞詮) 격국론, 궁통보감(窮通寶鑑) 조후론, 적천수(滴天髓) 강약론을 기반으로 한 학문적 사주 해석 스킬이다.

---

## 핵심 원칙 — 사주 해석의 절대 기준

명리학자들이 사주를 논리적으로 해독할 때 가장 중요하게 삼는 절대 기준은 **월령(月令) 기반의 격국 파악, 조후(기후적 환경)의 조절, 그리고 기신을 제어하여 그릇을 키우는 구격(救格)의 원리**다. 이 세 가지가 톱니바퀴처럼 맞물려 한 사람의 명(命)을 입체적으로 그린다.

### 1. 월령과 투출·통근의 중요성

- **월령은 사주의 강령**: "月令乃提綱之最重" — 월지를 가장 무겁게 다룬다.
- **투출(透出)**: 월지 장간이 천간에 드러나는 것. 투출이 있어야 格이 뚜렷해진다.
- **통근(通根)**: 천간의 오행이 지지에 같은 오행을 두어 힘을 얻는 것. **통근 없는 천간은 뿌리 없는 나무** — 겉으로 있어 보이지만 실력이 없다.

### 2. 격국론의 핵심 — 성패와 구응

格은 주어진 환경 속에서 어떻게 살아남고(용신), 무엇을 경계해야 하는지(기신)를 따지는 정밀한 시스템이다.

- **성격의 3기준**: ① 격국 핵심 오행이 월령 득기, ② 용신이 건재, ③ 기신이 없거나 완전히 제어됨
- **맑음과 혼탁**: 오행 관계가 단순·명확하면 맑은 사주, 관살혼잡 등으로 복잡하게 얽히면 혼탁
- **패중취귀(敗中取貴)**: "패격이라도 구신이 있으면 귀해질 수 있다" — 자평진전의 가장 중요한 통찰. 상관이 정관을 극하는 패격도 인성이 상관을 제어(상관패인)하면 관·상관·인성의 삼중 구조로 오히려 귀한 명이 된다.

### 3. 조후론의 핵심 — 계절과 용신의 역전

- **조후 역전의 원리**: 계절에 따라 같은 오행이 용신에서 기신으로, 또는 그 반대로 바뀐다. **어떤 오행도 절대적인 용신이나 기신이 될 수 없다.**
- **예시**: 한여름(午월)의 丙火에게 壬水는 조열함을 식히는 절대 용신이지만, 한겨울(子월)의 丙火에게 壬水는 약해진 화기를 꺼뜨리는 기신이 된다.

### 4. 학파 이견 처리 원칙

**→ 학파별 이견이 있는 12개 쟁점의 상세 정리는 `references/23_hakpa_igyeon.md`를 참조한다**

- **격국론 vs 억부론**: 格을 먼저 보고, 억부는 格 내에서 보조적으로 고려한다.
- **격국론 vs 조후론**: 格의 성패를 먼저 보고, 조후로 세부 보정하는 **병용 방식**을 취한다.
- **이견이 있는 경우**: 단정하지 않고 두 견해를 모두 제시하며 근거를 밝힌다.

## Step 1: 사주팔자·대운 계산 — 반드시 스크립트 실행

모든 결정론적 계산(연·월·일·시주, 절기, 십신, 장간, 통근, 대운, 세운·월운 간지)은 `scripts/saju_calc.py`가 담당한다. **산문 규칙으로 암산하지 않는다** (errata #1·#2·#3의 발생 지점이 전부 암산이었다).

```bash

스크립트가 처리하는 것: 진태양시 보정(기본 서울 -32분, `--no-solar-correction`으로 해제), 절기 기반 월주(태양 황경 실계산), 입춘 기준 연주, 자시 규칙(`--zasi next|same`), 대운 방향(**연간** 기준 양남음녀 순행)·대운수(절기까지 시간/3)·간지열, 통근 등급(록왕/생지/여기/묘고), 월지 장간 투출 체크.

**스크립트 출력 후 Claude가 확인할 것**:
- `경고` 필드: 절입 ±24시간 이내 출생, 서머타임 연도, 1954~61 UTC+8:30 — 해당 시 사용자에게 알리고 양쪽 가능성 제시
- 정각(xx:00) 출생: 이전 시지로 처리됨을 고지하고, 경계라면 양쪽 시주를 모두 언급

## Step 2: 기본 정보 해석

- 나를 극(克)하는 것: 관성 (음양 같으면 편관/칠살, 다르면 정관)

## Step 3-5: 강약·종격 판단 (격국 분석 전 선행)

**→ `references/21_jeokcheonsu_gangyak.md`를 참조한다**

격국 분석(Step 4)에 들어가기 전, 먼저 일간의 강약과 종격 여부를 판단한다.

```
판단 순서:
① 일간의 통근 수·질, 월령 득기 여부, 천간 생조 확인 → 신강/신약 판단
② 일간이 극약(통근 0~1, 월령 실기, 천간 생조 없음)이면 → 종격 여부 검토
③ 종격 성립 시 → 종격의 용신·기신 체계 적용 (Step 4 격국론 미적용)
④ 종격 불성립 시 → Step 4 격국론 진행

⚠️ 종격은 극소수(전체의 5~10% 이하)에만 해당.
   "신약"과 "종격"은 전혀 다른 판단이다. 남발 금지.
```

## Step 4: 격국 분석

**→ 반드시 `references/19_sungpae_rules.md`를 읽은 후 분석한다**

### 4-1. 格 결정 순서

> **핵심 원칙**: 格은 반드시 월지(月令)에서 출발한다. 천간에 떠 있는 글자가 아무리 강해 보여도, 월지 장간을 먼저 확인하기 전에는 格을 결정하지 않는다. 이것이 자평진전 격국론의 절대 규칙이다.

**① 월지 장간 투출 여부 확인 (최우선)**

```
월지 장간(본기·중기·여기) 중 천간에 투출한 것이 있는가?

[있으면]: 투출된 장간 → 일간과의 관계 → 십신 → 格 결정
           복수 투출 시: 본기에 가까울수록 格의 힘이 강함
           (단, 중기·여기라도 천간에 투출했으면 格 성립 가능)

[없으면]: → 아래 ②번으로
```

**② 월지 장간 미투출 시 → 월지 본기(本氣)로 格 결정**

```
월지 장간이 천간에 하나도 투출하지 않았을 때:
→ 월지의 본기(本氣, 가장 강한 장간)를 기준으로 格을 결정한다.
→ 천간에 떠 있는 다른 오행을 보고 格을 정하는 것은 오류다.

예) 월지 子水, 천간에 壬·癸 없음 → 子의 본기 癸水로 格 결정
    戊土 일간이면: 癸水(음수) vs 戊土(양토) → 음양 다르게 극 → 正財格
    ※ 천간의 丙火를 보고 편인격이라 하면 치명적 오류
```

**③ 格 결정 후 반드시 확인할 것**

```
- 格의 핵심 오행(월지 본기 or 투출 장간)이 지지에 통근하는가?

## Step 7: 대운 해석

대운 해석은 **원국의 格과 용신·기신 구조를 먼저 확정한 뒤**, 각 대운이 그 구조에 어떤 영향을 주는지를 판단한다. 대운 자체의 십신보다 **원국과의 상호작용**이 핵심이다.

### 7-1. 판단 순서

```
Step A: 대운 천간·지지의 십신 확인
Step B: 원국 용신·기신과의 관계 판단
Step C: 대운이 원국에 일으키는 합·충 확인
Step D: 종합 판정
```

### 7-2. Step A — 대운 십신 확인

대운 천간과 지지를 각각 일간 기준 십신으로 변환한다.
지지는 본기(本氣) 기준으로 십신을 판단한다.

### 7-3. Step B — 원국 용신·기신과의 관계

| 대운 십신의 역할 | 판정 | 설명 |
|---|---|---|
| 원국 **용신**과 같은 오행 | 길(吉) | 용신 강화 — 格의 성립 조건이 더 강해짐 |
| 원국 **희신**과 같은 오행 | 길(吉) | 용신을 생하는 흐름 강화 |
| 원국 **용신을 생**하는 오행 | 길(吉) | 간접적으로 用을 보조 |
| 원국 **기신**과 같은 오행 | 흉(凶) | 기신 강화 — 格을 파괴하는 힘이 세짐 |
| 원국 **기신을 생**하는 오행 | 흉(凶) | 간접적으로 기신을 돕는 흐름 |
| 원국 **기신을 극**하는 오행 | 길(吉) | 기신 제어 — 구격(救格) 역할 |
| 원국 **용신을 극**하는 오행 | 흉(凶) | 용신 파괴 |
| 중립 오행 | 중(中) | 원국 구조에 직접 영향 없음 |

**⚠️ 추가 판단 — 용신이 길해 보여도 흉으로 뒤집히는 두 가지 경우**:

**① 재다신약(財多身弱)**
- 재성(용신)이 대운·세운에서 강화될 때, 일간이 이를 감당할 힘이 있는지 반드시 확인한다.
- 일간의 통근(뿌리)이 약하거나 삼합·합거로 뿌리가 사라진 상태에서 재성이 폭발적으로 강해지면 → 재물이 오히려 일간을 압박하는 재다신약 패격.
- 판단 기준: **일간의 지지 통근처가 몇 개인가?** 1개 이하이고 재성이 3개 이상 활성화되면 재다신약 위험.

**② 통관(通關)의 기여 vs 과잉**
- 기신(식상)과 용신(정관) 사이에서 재성이 통관(기신의 기운을 흡수해 용신으로 전달) 역할을 할 때 → 재성은 구신(救神)으로 길.
- 그러나 통관 역할을 넘어 재성 자체가 과다해지면 → 일간 신약 + 재다 → 패격.
- **통관이 아름다운 것과, 재성이 과다한 것은 다르다.** 대운에서 재성 세력이 폭발할 때는 반드시 일간의 감당 능력을 함께 본다.


**병약론 상세** (설기태심·조토불생금·탐재괴인·상관패인·적용 프로세스): `references/25_byeongyak.md`

### 7-4. Step C — 합·충 확인

대운이 원국의 지지와 합·충을 일으킬 때 추가 판단이 필요하다.

**대운 지지가 원국 지지와 충(沖)할 때**:
- 충당하는 지지가 **기신**이면 → 기신 제거 → 길
- 충당하는 지지가 **용신의 뿌리(통근처)**이면 → 용신 약화 → 흉
- 충당하는 지지가 **중립**이면 → 변화·이동·불안정 (흉보다 변동으로 해석)

**대운 천간이 원국 천간과 합(合)할 때**:
- 합거되는 것이 **기신**이면 → 기신 합거 → 길
- 합거되는 것이 **용신**이면 → 용신 합거 → 흉
- 합거되는 것이 **구신(救神, 기신을 제어하던 오행)**이면 → 구신 합거 → 기신이 되살아나 패격 → **흉**
  ※ 이 경우가 특히 중요: 겉보기에 기신이 합거되는 것처럼 보여도, 실제로는 구신이 묶이는 구조일 수 있다.
- 합화(合化)의 결과 오행이 용신이면 → 길, 기신이면 → 흉

**대운 지지가 원국 지지와 삼합(三合)·방합(方合)을 이룰 때**:
- 합으로 형성된 오행이 용신이면 → 길
- 합으로 형성된 오행이 기신이면 → 흉
- ⚠️ **삼합에 묶인 지지가 일간의 통근처(뿌리)일 때 추가 주의**:
  → 해당 지지가 삼합으로 오행이 변질되면 일간의 뿌리가 사라진다.
  → 일간이 신약한 사주에서 유일한 통근처가 삼합으로 묶히면 일간이 뿌리를 잃어 관성·재성을 감당할 힘마저 상실.
  → 겉보기의 합충 길흉 판단에 앞서, **일간의 근기(根基)가 유지되는가**를 반드시 확인한다.

**⚠️ 월지 합거 특별 주의**:
- 월지는 格을 이루는 핵심이자 格국의 근거지다.
- 대운·세운의 지지가 월지와 합(육합·삼합)을 이루어 **월지의 오행이 합거·변화**될 때:
  → 이것은 단순한 '기신 제거'가 아니라 **格 자체의 근거가 흔들리는 사건**으로 본다.
  → 합거 결과의 오행이 무엇인지, 格에 유리한지 불리한지 반드시 판단한다.
  → 조후론적으로도 월지(계절)가 변형되는 의미이므로 조후 용신에 미치는 영향도 함께 검토한다.

### 7-5. Step D — 종합 판정 기준

**천간과 지지를 분리해서 판단한 뒤 종합**한다.
천간은 상반기(1~5년), 지지는 하반기(6~10년)에 더 강하게 작용하는 경향이 있다.

| 천간 | 지지 | 종합 판정 |
|---|---|---|
| 길 | 길 | 대길(大吉) — 10년 내내 순조로움 |
| 길 | 흉 | 선길후흉(先吉後凶) — 전반부 좋고 후반부 어려움 |
| 흉 | 길 | 선흉후길(先凶後吉) — 전반부 어렵고 후반부 회복 |
| 흉 | 흉 | 대흉(大凶) — 용신 파괴 또는 기신 강화 지속 |
| 길 | 중 | 길(吉) — 전반 좋고 후반 평범 |
| 중 | 길 | 길(吉) — 전반 평범하고 후반 좋음 |
| 흉 | 중 | 흉(凶) — 전반 어렵고 후반 평범 |
| 중 | 흉 | 흉(凶) — 전반 평범하고 후반 어려움 |
| 중 | 중 | 평(平) — 큰 변화 없이 평이한 시기 |

### 7-7. 格별 용신·기신 빠른 참조

대운 해석 시 원국 格의 용신·기신을 아래 표로 빠르게 확인한다.

| 格 | 용신(길) | 희신(길) | 기신(흉) | 구신(救神) |
|---|---|---|---|---|
| 정관격 | 재성, 인성 | 비겁(신약시) | 상관, 칠살, 겁재 | 인성(상관패인), 관성(겁재제어) |
| 칠살격 | 식신, 인성 | — | 재성(재생살), 상관(불안정) | 식신(제살), 인성(화살) |
| 정인격 | 관성 | 비겁 | 재성(재파인) | 비겁(재성제어) |
| 편인격 | 관성(칠살) | — | 식신(효신도식) | 재성(편인제어) |
| 정재격 | **식상(식상생재)**, 관성(재생관) | — | 겁재·비견(겁재탈재) | 관성(겁재제어) |
| 편재격 | **식상(식상생재)**, 관성(재생관) | — | 겁재·비견(겁재탈재), 칠살(재생살) | 관성(겁재제어) |
| 식신격 | 재성(식신생재), 칠살(식신제살) | — | 편인(효신도식) | 재성(편인제어) |
| 상관격 | 재성 또는 인성 | — | 정관(상관견관), 재성(재파인시) | 인성(상관패인) |

**⚠️ 정재격·편재격에서 식상의 역할 — 특별 주의**:
- 재격에서 식상(食傷)은 **재성을 생하는 용신(식상생재)**이다. 절대 기신이 아니다.
- 편인격에서 식신이 기신인 것(효신도식)과 혼동하지 말 것.
- 재격에서 식상 대운은 길운으로 판단한다.

**⚠️ 구신(救神) 합거 — 반드시 확인**:
- 구신은 기신을 제어하여 格을 살리는 오행이다.
- 대운·세운이 구신을 합거하면 → 기신이 되살아나 패격 → 흉
- 예) 정재격에서 겁재(기신)를 관성(구신)이 제어 중일 때, 대운에서 관성이 합거되면 → 겁재탈재 재발 → 흉

**⚠️ 조후 용신 ≠ 격국 용신일 때**:
- 격국 용신과 조후 용신이 다를 수 있다.
- 조후 용신이 격국의 기신일 경우: 格의 성패를 먼저 보고 조후는 보조로 참고한다.
- 조후 용신이 格에도 유익할 경우(겸용): 해당 오행의 가치가 더욱 높아진다.
- 어느 대운이든, 격국과 조후 **양쪽에서** 그 오행의 역할을 반드시 검토한다.

※ 위 표는 일반 원칙이며, 원국의 구체적 구조·통근 여부·합충에 따라 달라진다. 반드시 Step 4(격국 분석) 결과를 먼저 확인한 뒤 이 표를 보조 참조로 사용할 것.


대운 해석 출력은 `references/27_output_formats.md` 참조.

## Step 8: 세운(歲運) 분석

세운은 해당 연도의 천간·지지가 원국 + 현재 대운과 **동시에** 상호작용하는 층위다. 글자가 원국(8자) + 대운(2자) + 세운(2자) = 최대 12자로 늘어나므로, 합·충의 경우의 수가 많아진다. 핵심은 **세운이 대운의 흐름을 가속하는지, 역전시키는지**를 판단하는 것이다.


### 8-1. 세운 간지 산출
`saju_calc.py --year-fortune <연도>`로 세운 간지와 12개월 절입표를 산출한다.

### 8-2. 세운 해석 순서

```
Step A: 세운 천간·지지의 십신 확인 (일간 기준)
Step B: 원국 용신·기신과의 관계 판단 (대운 해석 7-3과 동일한 기준)
Step C: 세운이 원국·대운에 일으키는 합·충 확인
         → 원국 글자와의 합충 + 대운 글자와의 합충 모두 검토
Step D: 대운과의 관계 판단
         → 세운이 대운 흐름을 강화하는가, 역전시키는가
Step E: 종합 판정
```

### 8-3. 세운-대운 관계 판단

| 대운 | 세운 | 해석 |
|---|---|---|
| 길운 대운 | 세운도 길 | 해당 연도 최대 길 — 용신 이중 강화 |
| 길운 대운 | 세운이 흉 | 대운의 길함이 일시 방해 — 그해만 어려움 |
| 흉운 대운 | 세운도 흉 | 해당 연도 최대 흉 — 기신 이중 강화, 극히 주의 |
| 흉운 대운 | 세운이 길 | 대운의 흉함이 일시 완화 — 그해만 숨통 |

### 8-4. 글자 수 증가에 따른 합충 처리 원칙

세운에서는 원국 8자 + 대운 2자 + 세운 2자 = 최대 12자가 동시에 작용한다.

- **우선순위**: 천간합 > 지지충 > 삼합·방합 순으로 강도가 강하다.
- **중복 합충**: 같은 글자에 합과 충이 동시에 발생하면, 충이 합을 깨는 경향이 있다(충산합해, 沖散合解).
- **세운 천간합이 대운·원국 글자를 합거**할 때: 합거 대상이 용신이면 흉, 기신이면 길.
- **세운 지지가 대운 지지와 충**할 때: 대운 지지의 흉함을 깨면 길, 대운 지지의 길함을 깨면 흉.
- **삼합 완성**: 원국·대운·세운 세 글자가 삼합을 이루면 해당 오행이 매우 강하게 작용.

**⚠️ 천간·지지의 질적 상호작용 — 단순 길흉을 넘어선 판단**

천간과 지지가 같은 방향이어도, 지지의 성질이 천간의 기능을 훼손하는 경우가 있다.

**대표 사례: 壬水가 辰土와 함께 올 때 (壬辰월·壬辰년)**

```
표면: 壬(희신·수생목) + 辰(土·기신) → 천간 길, 지지 흉 → 중(中) 판정
실제: 壬水(맑은 물)가 辰土(진흙)와 만나면 물이 탁해지는 현상 발생
      → 기토탁임(己土濁壬)과 유사: 土가 水의 순수한 기능을 혼탁하게 만듦
      → 壬의 수생목(용신 보강) 효과가 약화됨
```

**실전 해석 원칙**:
- 희신 천간 + 기신 지지 조합은 **단순 중립이 아니라 '기능 혼탁'**으로 읽는다.
- 물(壬·癸)이 土(辰·丑·未·戌)와 함께 올 때: 水의 흐름이 탁해지고 정보·판단·소통이 혼란스러워질 수 있다.
- 金(庚·辛)이 火(巳·午·戌)와 함께 올 때: 金이 녹을 위험 — 재성이나 인성이 약화.
- **월 단위 분할 적용**: 천간(월초~중순)과 지지(월말) 기운의 강약 차이를 활용한다.
  → 천간 희신이 강한 월초·중순: 중요 결정·계약·추진에 유리
  → 지지 기신이 강해지는 월말: 무리한 확장보다 확보한 것을 공고히 다지기


세운 출력 형식은 `references/27_output_formats.md` 참조.

## Step 9: 월운(月運) 분석

월운은 해당 월의 월주 간지가 원국 + 대운 + 세운과 **동시에** 작용하는 층위다. 글자가 최대 14자(원국8 + 대운2 + 세운2 + 월운2)로 늘어나며, 합충의 경우의 수가 더욱 많아진다. 월운은 세운의 흐름 안에서 **구체적인 사건·변동이 일어나는 시점**을 포착한다.


### 9-1. 월운 간지 산출
`saju_calc.py --year-fortune <연도>` 출력의 월별 간지·절입 시각을 사용한다.

### 9-2. 월운 해석 순서

```
Step A: 월운 천간·지지의 십신 확인
Step B: 원국 용신·기신과의 관계 (7-3과 동일 기준)
Step C: 합충 확인 — 원국·대운·세운·월운 글자 전체를 대상으로
Step D: 세운 흐름과의 관계
         → 세운의 기조를 강화하는 달인가, 완화하는 달인가
Step E: 종합 판정
```

### 9-3. 글자 수 증가 처리 원칙

월운에서는 최대 14자가 동시 작용한다. 이 때 다음 원칙을 따른다.

- **층위별 강도**: 원국(뿌리) > 대운(10년) > 세운(1년) > 월운(1개월) 순으로 영향력이 강하다. 월운은 강도가 약하지만 **발현 시점(timing)**을 결정한다.
- **세운의 기조 안에서 월운을 읽는다**: 흉운 세운에서 길운 월운은 '잠깐 숨통'이지, 흉을 완전히 뒤집지 않는다. 반대로 길운 세운에서 흉운 월운은 일시적 방해이지 전체 흐름을 바꾸지 않는다.
- **결정적 합충**: 세운·월운 두 글자가 동시에 원국의 같은 글자와 합·충을 일으킬 때 → 해당 월에 사건이 집중적으로 발생하는 경향.
- **삼합·방합 완성 월**: 원국·대운·세운 글자와 월운 글자가 삼합을 완성하는 달 → 해당 오행이 최대로 작용하는 시점.


월운 출력 형식은 `references/27_output_formats.md` 참조.

- **절기 경계일**: 절기 당일 출생자는 절기 전후를 명시하고, 정확한 절입 시각을 모를 경우 양쪽 가능성을 제시한다.
- **학파 이견**: 격국론(자평진전)·강약론(적천수)·조후론(궁통보감)이 충돌할 경우 두 견해를 모두 제시하고, 어느 쪽이 더 강한 근거를 가지는지 판단한다. 상세 이견 목록은 `references/23_hakpa_igyeon.md` 참조.
- **종격 판단**: 종격은 극소수에만 해당하며 남발하지 않는다. "신약"과 "종격"은 전혀 다른 판단이다. 상세 기준은 `references/21_jeokcheonsu_gangyak.md` 참조.
- **합충형파해**: 합·충을 핵심으로 분석하고, 형·파·해는 보조 참조한다. 상세 규칙은 `references/22_hapchung_shinsal.md` 참조.
- **시간 미상**: 시주를 모를 경우 시주 제외 분석 후, 가능한 시주 범위를 제시한다.
- **음력 입력**: 음력 날짜를 입력받으면 양력으로 변환 후 계산한다. 음력→양력 변환은 만세력 기준이며, 불확실한 경우 명시한다.
