# 자평진전(子平眞詮) — 편인격(偏印格)

> 자평진전 격국론 시리즈 4편. 편인격은 정인격보다 편벽하고 특수한 성질을 가진다.

---

## 1. 정의와 본질

편인격(偏印格)은 **월지 장간 중 편인(偏印)이 천간에 투출**하여 성립한다.
편인(偏印)이란 일간을 음양이 같게 생(生)하는 오행이다.

| 일간 | 편인 |
|---|---|
| 甲(양목) | 壬(양수) |
| 乙(음목) | 癸(음수) |
| 丙(양화) | 甲(양목) |
| 丁(음화) | 乙(음목) |
| 戊(양토) | 丙(양화) |
| 己(음토) | 丁(음화) |
| 庚(양금) | 戊(양토) |
| 辛(음금) | 己(음토) |
| 壬(양수) | 庚(양금) |
| 癸(음수) | 辛(음금) |

**편인의 본질**: 정인이 따뜻하고 정통적인 어머니라면, 편인은 계모·이모처럼 편벽하고 특수한 생조다. 기예(技藝)·종교·철학·특수 학문과 연결된다.

**편인의 가장 중요한 특성 — 효신도식(梟神盜食)**

> "편인이 식신을 극한다 — 효신도식(梟神盜食)"

효(梟)는 올빼미를 뜻하며, 어미를 잡아먹는 새로 비유된다. 편인이 식신을 극하면 식신의 생산·표현 기능을 완전히 박탈한다. 이것이 편인격의 가장 독특한 특성이다.

---

## 2. 성립 조건

월지 장간 중 편인이 천간에 투출하면 격이 성립한다.

**편인격과 정인격의 구별**: 편인과 정인이 함께 있으면 인성혼잡(印星混雜). 이 경우 격의 순수성이 손상된다.

---

## 3. 용신·희신·기신

### 용신

**① 관성(官星·殺星) — 관인상생·살인상생**

편인격도 정인격처럼 관성이 인성을 생하는 구조를 선호한다. 다만 편인격은 칠살(편관)과의 살인상생(殺印相生)도 유효하다.

```
살인상생 구조:
칠살(殺) →생→ 편인(偏印) →생→ 일간(我)
```

**② 재성(財星) — 재제편인(財制偏印)**

정인격에서 재성은 최대 기신이었지만, 편인격에서 재성은 편인을 제어하는 용신이 될 수 있다. 편인이 너무 강할 때 재성이 편인을 극(財克印)하여 균형을 잡는다.

### 기신

**① 식신(食神) — 효신도식의 역방향**

편인격에서 식신이 있으면 편인이 식신을 극하여 식신의 기능이 박탈된다. 그러나 이 상황은 편인격 자체의 문제라기보다 편인이 식신을 만났을 때의 부작용이다.

**② 편인 과다(偏印過多)**

편인이 너무 많으면 일간이 나태하고 의존적이 되며, 편벽한 성질이 더욱 강해진다.

---

## 4. 성격(成格)

### 유형 ① 살인상생(殺印相生)

```
예시: 甲木 일간, 子월(癸水 편인 투출), 천간에 庚金 칠살
      庚金이 癸水를 생(金生水) → 癸水가 甲木을 생(水生木)
      → 살인상생 완성
```

무관(武官) 또는 특수 권력 구조의 명에서 자주 나타난다.

### 유형 ② 재제편인(財制偏印)

```
예시: 편인이 강한 구조에서 재성이 편인을 적절히 극
      → 편인의 편벽함이 조절되어 격이 균형을 이룸
```

---

## 5. 패격(敗格)

### 유형 ① 효신도식(梟神盜食)

```
예시: 甲木 일간, 子월(癸水 편인), 천간에 丙火 식신
      癸水 편인이 丙火 식신을 극(水克火) → 식신 기능 박탈
      → 표현·생산·복록이 모두 손상 → 패격
```

### 유형 ② 편인과다신약(偏印過多身弱)

편인이 너무 많으면 일간이 과도하게 의존적이 되어 아무것도 이루지 못하는 나태함이 생긴다.

---

## 6. 구격(救格)

### 유형 ① 재제편인(財制偏印)으로 식신 보호

효신도식 상황에서 재성이 편인을 제어하면 식신이 살아난다.

```
예시: 癸水 편인이 丙火 식신을 극하려 할 때
      戊土 편재가 癸水를 극(土克水) → 편인 제어 → 식신 보호 → 구격
```

---

## 7. 편인격의 특수 성질

**기예명(技藝命)**: 편인격은 정통 학문보다 기예·예술·종교·철학과 연결된다. 정인격이 정통 교육으로 관직에 오르는 형상이라면, 편인격은 특수한 재능으로 두각을 나타내는 형상이다.

**의식주 문제**: 효신도식이 발생하면 식신(의식주·복록)이 박탈되어 생계 문제가 생기기 쉽다고 본다.

---

## 8. 정인격 vs 편인격 비교

| | 정인격 | 편인격 |
|---|---|---|
| 인성의 성질 | 따뜻하고 정통적 | 편벽하고 특수 |
| 재성 역할 | 최대 기신(재파인) | 용신 가능(재제편인) |
| 식신과의 관계 | 큰 충돌 없음 | 효신도식 위험 |
| 성격 키워드 | 학문, 명예, 정통 | 기예, 종교, 특수 재능 |
| 살과의 관계 | 관인상생 선호 | 살인상생도 유효 |

---

## 9. 학파 간 이견

**이견 ①** 재성의 역할: 자평진전은 편인격에서 재성을 편인 제어의 용신으로 볼 수 있다고 하지만, 정인격의 재파인 원칙을 편인격에도 동일하게 적용해야 한다는 견해가 있다.

**이견 ②** 효신도식의 심각성: 일부 학파는 효신도식을 반드시 흉하게 보지 않고, 편인과 식신의 세력 비교에 따라 판단해야 한다고 본다.

---

## 10. 편인격 핵심 요약

핵심 특성: 효신도식(편인이 식신을 극)이 가장 독특한 패턴
성격: 살인상생 또는 재제편인으로 편인 조절
패격: 효신도식(식신 박탈) 또는 편인 과다
구격: 재성이 편인을 제어하여 식신 보호
