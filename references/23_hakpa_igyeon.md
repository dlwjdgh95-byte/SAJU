# 학파별 이견 정리 — 자평진전파 vs 적천수파 vs 조후파

> 이 문서는 명리학 3대 학파(자평진전파·적천수파·조후파)가 서로 다른 견해를 보이는 주요 쟁점을 정리한 것이다.
> 사주 해석 시 학파 간 이견이 있는 항목을 만나면, 이 문서를 참조하여 양쪽(또는 세 쪽) 견해를 모두 제시한다.

---

## 학파 개요

```
자평진전파 (격국론 중심):
- 경전: 자평진전(子平眞詮), 심효첨(沈孝瞻) 저
- 핵심: 월령 기반 격국 판단 → 성격·패격·구격의 체계
- 방법론: 투출·통근 → 격국 결정 → 용신·기신 → 성패 판단
- 강점: 논리적·체계적, 재현 가능한 분석 프레임워크

적천수파 (강약론·종격론 중심):
- 경전: 적천수(滴天髓), 경도(京圖) 저 / 임철초(任鐵樵) 주석
- 핵심: 일간 강약 판단 → 억부(抑扶)·종격 판단
- 방법론: 일간의 세력 대비 → 신강이면 극설, 신약이면 생조
- 강점: 직관적, 종격·특수격 판단에 유리

조후파 (조후론 중심):
- 경전: 궁통보감(窮通寶鑑) / 난강망(欄江網)
- 핵심: 계절(월령)의 한난조습에 따른 조후 용신 결정
- 방법론: 일간 + 출생 월 → 필요한 오행(조후 용신) 결정
- 강점: 계절감에 충실, 실전 적중률이 높다는 경험론적 주장
```

## 이견 항목 1: 용신(用神) 결정의 최우선 기준

### 쟁점: 격국·억부·조후 중 무엇이 우선인가?

```
자평진전파:
- 격국 용신이 최우선
- "月令用神은 格을 이루는 핵심이며, 用神은 格에서 나온다"
- 억부론은 격국 안에서 보조적으로만 사용
- 조후는 격국·용신 결정 후 세부 보정 단계에서 참고

적천수파:
- 억부(抑扶) 용신이 최우선
- "일간의 강약을 먼저 판단하고, 강하면 극설하는 오행이 용신, 약하면 생조하는 오행이 용신"
- 격국은 일간 강약 판단 이후에 참고
- 종격 판단이 격국보다 앞선다

조후파:
- 조후 용신이 최우선
- "한겨울의 丙火에게는 무엇보다 따뜻함(木·火)이 필요하고, 한여름의 壬水에게는 시원함이 필요하다"
- 계절의 한난조습을 조절하는 오행이 곧 용신
- 격국·억부는 조후 용신을 보조

본 스킬의 입장:
→ 격국론 우선, 조후론으로 보정하는 병용 방식
→ 단, 종격 여부는 격국 판단 이전에 먼저 가린다 (적천수파 수용)
→ 조후 용신과 격국 용신이 일치하면 최상, 불일치하면 격국 우선
```

## 이견 항목 2: 종격(從格) 성립 기준의 엄격성

### 쟁점: 미약한 통근이 있을 때 종격으로 볼 것인가?

```
적천수파 (관대한 기준):
- 통근이 1개 있어도, 그 통근이 충·합거·조토 위의 약한 근이면 종격으로 봄
- "뿌리가 있어도 기능하지 못하면 없는 것과 같다"
- 가종(假從)이라는 중간 개념을 활발히 사용
- 종격의 비율을 상대적으로 높게 봄

자평진전파 (엄격한 기준):
- 통근이 1개라도 살아 있으면 종격이 아니라 극약 일반격
- "뿌리가 있으면 일간은 무너지지 않는다"
- 종격의 비율을 매우 낮게 봄 (전체 사주의 5% 이하)
- 가종 개념을 인정하지 않거나 회의적

조후파:
- 종격 판단에 대해 독자적 기준이 뚜렷하지 않음
- 주로 적천수파나 자평진전파의 기준을 차용
- 조후적으로 볼 때, 종격이든 일반격이든 계절 조절이 중요하다는 입장

실전 처리 원칙:
→ 미약한 통근이 있는 경계 사례에서는 양쪽 견해를 모두 제시
→ 대운의 실제 흐름으로 검증 가능: 인성·비겁 대운에서의 길흉이 판별 기준
```

## 이견 항목 3: 양인격(羊刃格)의 독립 격국 인정 여부

### 쟁점: 양인을 독립된 격국으로 볼 것인가?

```
자평진전파:
- 양인격을 독립 격국으로 인정
- "양인합살(羊刃合殺)" — 양인이 있으면 칠살로 제어해야 성격
- 양인격의 용신: 칠살(관살)
- 양인격의 기신: 재성(양인의 기운을 더 부추김)

적천수파:
- 양인을 독립 격국으로 보지 않음
- 양인은 신살(神煞)의 일종이며, 일간이 극강한 상태의 지표일 뿐
- 일간이 강하면 식상·재성·관살로 억부 → 별도의 양인격 불필요

조후파:
- 양인격에 대해 별도 논의 없음
- 양인이 있는 사주도 조후 용신 결정에는 영향 없음

실전 처리 원칙:
→ 양인이 월지에 있는 사주에서 자평진전파의 양인격론을 기본으로 제시
→ 적천수파의 견해(독립 격국 부인)도 함께 언급
→ 양인의 실전 영향력은 학파를 불문하고 인정 (과격·사고·수술의 상)
```

## 이견 항목 4: 격국론에서 억부론의 위치

### 쟁점: 신강·신약(억부론)을 격국 판단에 어떻게 반영하는가?

```
자평진전파:
- 격국론이 주(主), 억부론은 종(從)
- 格의 성패를 먼저 보고, 格 안에서 신강·신약을 보조적으로 고려
- "신약하다고 무조건 인성이 용신인 것은 아니다 — 格이 요구하는 용신이 다를 수 있다"
- 예) 정재격 신약: 인성이 아니라 식상(식상생재)이 用일 수 있음 (格의 논리)

적천수파:
- 억부론이 주(主), 격국론은 참고
- 일간의 강약을 먼저 판단하고, 강약에 따라 용신을 결정
- "신약하면 인성·비겁, 신강하면 식상·재성·관살"이 기본 공식
- 격국은 억부 판단 이후 해석의 풍부함을 위해 참고

조후파:
- 억부론과 조후론을 병용
- 신강·신약 판단 후, 조후로 최종 용신을 확정
- "신약해도 조후가 더 급하면 조후 용신을 우선한다"

이견의 핵심:
→ 자평진전파: 格이 먼저, 신강·신약은 格 안에서
→ 적천수파: 신강·신약이 먼저, 格은 이후
→ 이 차이 때문에 같은 사주에서 용신이 달라질 수 있음
```

## 이견 항목 5: 조토(燥土)와 습토(濕土)의 오행 기능 차이

### 쟁점: 未·戌(조토)과 丑·辰(습토)을 같은 土로 볼 것인가?

```
적천수파 (구분 중시):
- 조토와 습토는 같은 土지만 기능이 전혀 다름
- "조토불생금(燥土不生金)" — 未·戌의 건조한 흙은 金을 생하지 못함
- "습토불극수(濕土不克水)" — 丑·辰의 축축한 흙은 水를 제대로 극하지 못함
- 통근 판단에서도 조토·습토를 구분: 조토 위의 金은 약한 근

자평진전파 (상대적으로 덜 구분):
- 土는 土: 장간의 본기가 중요하지 조토·습토 구분은 부차적
- 未의 본기는 己土, 戌의 본기는 戊土 → 장간으로 판단하면 됨
- 계절에 따른 오행 기능 변화를 인정하지만, 이것을 격국 판단의 핵심으로 삼지는 않음

조후파 (강하게 구분):
- 조후론의 핵심이 한난조습이므로, 조토·습토 구분은 필수
- 여름 사주에서 조토가 많으면 → 조열함 가중 → 水가 더 급히 필요
- 겨울 사주에서 습토가 많으면 → 한습함 가중 → 火가 더 급히 필요

실전 처리 원칙:
→ 조토·습토 구분을 적용 (적천수파·조후파의 견해 수용)
→ 특히 金의 통근처 판단, 조후 용신 판단에서 반드시 구분
→ 격국 판단에서는 장간 본기를 우선하되, 조습의 질적 차이를 보조적으로 반영
```

## 이견 항목 6: 상관견관(傷官見官)의 해석

### 쟁점: 상관이 정관을 만나면 반드시 흉한가?

```
자평진전파:
- "상관견관 위환백단(傷官見官 爲禍百端)" — 상관이 관을 보면 화가 백가지
- 상관격에서 정관은 절대 기신: 상관이 관을 극하면 패격
- 단, 인성이 상관을 제어하면(상관패인) 구격 가능

적천수파:
- 상관견관을 절대적 흉으로 보지 않음
- 일간이 신강하고 관성이 약할 때, 상관이 관을 극하는 것은 오히려 밸런스
- 일간의 강약에 따라 상관견관의 길흉이 달라진다는 입장
- "신강한 사주에서 상관견관은 기운의 흐름이 원활한 것"

조후파:
- 상관견관 자체보다 조후 균형이 중요
- 상관과 관의 오행이 조후적으로 필요한 오행이면 흉하지 않을 수 있음

실전 처리 원칙:
→ 기본적으로 자평진전파의 "상관견관 흉"을 원칙으로 삼음
→ 단, 일간 강약과 전체 구조를 고려하여 적천수파의 견해도 부기
→ 상관패인(인성이 상관을 제어)이 있으면 구격으로 판단
```

## 이견 항목 7: 합충형파해의 실전 비중

### 쟁점: 형(刑)·파(破)·해(害)를 얼마나 중시할 것인가?

```
전통 명리파 (연해자평 계열):
- 합충형파해를 모두 동등하게 중시
- 특히 형(刑)은 관재·형벌의 핵심 지표
- 삼형이 있으면 반드시 법률 문제가 있다고 봄

적천수파:
- 충(沖)을 가장 중시, 형은 그 다음
- 파·해는 비중이 낮음
- "충은 실제 사건, 형은 갈등의 방향, 파·해는 미세한 불화"

자평진전파:
- 합(合)과 충(沖)을 중시 — 격국의 성패에 직접 영향
- 형·파·해는 격국 분석에 큰 영향을 주지 않으므로 비중 낮음
- "격국의 성패는 합충으로 판단하고, 형파해는 참고"

현대 실전파:
- 합 > 충 > 형 순으로 비중
- 파·해는 거의 무시하거나 참고 수준
- 삼형은 건강·관재와 관련하여 부분적으로 인정

실전 처리 원칙:
→ 합·충을 핵심으로 분석 (격국 성패에 직접 영향)
→ 형은 관재·건강 지표로 참고
→ 파·해는 인간관계 갈등의 보조 지표로만 언급
```

## 이견 항목 8: 신살(神煞)의 위상

### 쟁점: 신살을 사주 분석에 포함할 것인가?

```
자평진전파:
- 신살 거의 무시
- "격국과 용신이 사주의 전부이며, 신살은 미신에 가깝다"
- 양인만 예외적으로 격국론에 포함 (양인격)

적천수파:
- 신살을 참고 수준으로 사용
- 양인·역마·도화 정도만 실전에서 활용
- "신살은 象(상)을 풍부하게 하는 보조 도구"

조후파:
- 신살에 대한 별도 견해 없음
- 조후 분석과 신살은 독립적

전통 명리파 (연해자평·삼명통회 계열):
- 신살을 매우 중시
- 천을귀인·문창·공망·화개 등을 격국과 함께 분석
- "귀인이 있으면 패격도 구원받는다"

실전 처리 원칙:
→ 격국·용신 분석이 주, 신살은 부
→ 양인·역마·공망은 상대적으로 비중 높게 참고
→ 나머지 신살은 의뢰인이 물을 때 설명하는 수준
→ 신살만으로 길흉을 단정하지 않음
```

## 이견 항목 9: 공망(空亡)의 효력

### 쟁점: 공망에 해당하는 지지는 무력한가?

```
공망 중시파 (전통 명리·일부 적천수 해석자):
- 공망 지지는 힘이 없다 — 통근처로 인정하지 않음
- 공망에 든 용신은 힘을 잃으므로 길함이 줄어듦
- 공망에 든 기신은 흉함이 줄어드므로 오히려 길
- "공망은 비어 있는 것 — 뜻은 있으나 실현이 안 됨"

공망 경시파 (자평진전파·현대 실전파):
- 공망의 효력을 인정하지 않거나 매우 약하게 봄
- "대운에서 공망이 채워지면 정상 작동"
- 격국 분석에서 공망을 고려하지 않음
- "공망은 고대 점술의 잔재이며 논리적 근거 부족"

절충파:
- 원국 공망은 효력이 약하고, 대운 공망은 효력이 있다
- 또는 년지·일지의 공망만 인정하고 월지·시지는 무시

실전 처리 원칙:
→ 공망은 참고 수준으로 언급
→ 격국 판단에서 공망을 통근 무효의 근거로 삼지 않음
→ 의뢰인이 공망에 대해 물으면 학파 이견을 설명
```

## 이견 항목 10: 대운 천간·지지의 작용 시기

### 쟁점: 대운 천간은 전반 5년, 지지는 후반 5년에 작용하는가?

```
분리 작용설 (주류 전통):
- 대운 천간: 1~5년차에 주로 작용
- 대운 지지: 6~10년차에 주로 작용
- 선길후흉(先吉後凶), 선흉후길(先凶後吉) 판단의 근거

동시 작용설 (일부 현대파):
- 대운 천간·지지가 10년 내내 동시에 작용
- 전후반 구분은 경험적 근거 부족
- "대운은 10년 전체의 기운이며 분리할 수 없다"

절충설:
- 천간·지지가 10년 내내 작용하지만, 전반부에 천간의 영향이 상대적으로 강하고 후반부에 지지의 영향이 강해짐
- 완전한 분리가 아닌 비중의 이동

실전 처리 원칙:
→ 분리 작용설을 기본으로 채택 (선길후흉 등 판단에 유용)
→ 단, 절충설의 관점도 반영: "천간은 전반부에 더 강하게, 지지는 후반부에 더 강하게 작용하는 경향"
→ 세운과의 상호작용에서 세밀한 시기 판단 가능
```

## 이견 항목 11: 월지 육합의 격국 변화 여부

### 쟁점: 월지가 육합으로 합화되면 격국이 바뀌는가?

```
격국 변화 인정파 (일부 자평진전 해석자):
- 월지가 육합하여 합화가 성립하면, 월지의 오행이 변함
- 오행이 변하면 격국도 변함
- 예) 月子 + 日丑 → 子丑합土 → 월지가 土로 변하면 격국 재결정

격국 변화 부인파 (주류):
- 월지는 格의 근거지이므로 합으로 쉽게 변하지 않음
- 육합이 있어도 월지의 본기(本氣)는 유지됨
- 合化가 완벽히 성립하는 경우에만 격국 변화 고려 (극히 드뭄)

적천수파:
- 합화의 성립 조건을 엄격히 적용
- 월령의 지지를 받아야 합화 → 월지 자체가 합하는 경우 합화 판단이 복잡
- "월지의 합은 격국을 흔들 수 있지만, 합화까지 가야 격국이 변한다"

실전 처리 원칙:
→ 월지 육합은 격국을 쉽게 변화시키지 않는다 (부인파 기본)
→ 단, 합화 조건이 완벽히 갖춰지면 격국 변화 가능성을 제시
→ 대운·세운에서 월지 합이 발생하면 → 격국의 근거 동요로 해석
```

## 이견 항목 12: 육합 합화 결과 오행의 논쟁

### 쟁점: 卯戌합은 火인가? 午未합은 火인가 土인가?

```
卯戌합:
- 합화 火설 (주류): 卯戌합의 결과 오행은 火
- 합화 불성립설 (일부): 卯(木)와 戌(土)의 합은 합거일 뿐 합화되지 않음
- 적천수파: 합화가 되려면 월령이 火(巳午)여야 함

午未합:
- 합화 火설: 午(火) + 未(土지만 丁火 장간) → 火로 합화
- 합화 土설: 午의 己土 장간 + 未의 己土 본기 → 土로 합화
- 학파마다 견해가 갈리며 통일되지 않음

실전 처리 원칙:
→ 육합의 합화 결과가 논쟁적인 경우, 합거(合去)로만 판단
→ 합화 오행이 명확한 것(子丑→土, 寅亥→木, 辰酉→金, 巳申→水)만 합화 적용
→ 卯戌·午未의 합화 오행은 학파 이견이 있음을 명시
```
