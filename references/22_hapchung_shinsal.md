# 합충형파해(合沖刑破害)·신살(神煞) 규칙 정리

> 이 문서는 지지 간의 상호작용(합·충·형·파·해)과 주요 신살의 판단 규칙을 체계적으로 정리한 것이다.
> 격국 분석·대운 해석 시 합충 확인(Step 3, Step 7-4)의 세부 참조 자료로 사용한다.

---

## 1. 천간합(天干合)

### 1-1. 5종 천간합

| 합 | 합화(化) 오행 | 합화 조건 |
|---|---|---|
| 甲己합 | 土 | 월령이 土(辰戌丑未)이고 다른 방해 없을 때 |
| 乙庚합 | 金 | 월령이 金(申酉)이고 다른 방해 없을 때 |
| 丙辛합 | 水 | 월령이 水(亥子)이고 다른 방해 없을 때 |
| 丁壬합 | 木 | 월령이 木(寅卯)이고 다른 방해 없을 때 |
| 戊癸합 | 火 | 월령이 火(巳午)이고 다른 방해 없을 때 |

### 1-2. 천간합의 3가지 결과

```
① 합거(合去): 합한 두 글자가 본래 기능을 잃음
   - 가장 흔한 결과
   - 용신이 합거되면 흉, 기신이 합거되면 길
   - 인접한 두 천간 사이에서만 합이 성립 (격위합 주의)

② 합화(合化): 합하여 새로운 오행으로 변함
   - 성립 조건이 까다로움: 월령의 지지가 합화 오행을 지원해야 함
   - 합화가 되면 원래 오행 기능은 완전 소멸
   - ⚠️ 실전에서 합화는 드물다 — 합거가 기본

③ 합이불화(合而不化): 합하지만 변하지 않음
   - 합의 끌어당김은 있으나 오행 변화 없음
   - 두 글자의 기능이 약해지는 정도
```

### 1-3. 천간합 판단 시 주의사항

```
격위합(隔位合):
- 두 천간 사이에 다른 천간이 끼어 있으면 합의 힘이 약해짐
- 예) 년간 甲 — 월간 庚 — 일간 己
  → 甲己합이 있으나 庚이 가로막아 합력 약화
  → 학파에 따라 "격위합 불성립" 또는 "약한 합" 견해 차이

쟁합(爭合):
- 하나의 천간에 두 개의 합 대상이 있을 때
- 예) 甲 — 己 — 甲: 두 甲이 하나의 己를 두고 쟁합
  → 쟁합이면 합이 성립하지 않는다는 견해가 주류
  → 또는 인접한 쪽이 합하고 먼 쪽은 불합
```

## 2. 지지합(地支合)

### 2-1. 지지육합(六合)

| 합 | 합화 오행 | 비고 |
|---|---|---|
| 子丑합 | 土 | 동지·소한의 만남 — 합토 |
| 寅亥합 | 木 | 봄의 시작 — 합목 |
| 卯戌합 | 火 | 합화 논쟁 있음 (아래 학파 이견) |
| 辰酉합 | 金 | 합금 |
| 巳申합 | 水 | 합수 |
| 午未합 | 火 또는 土 | 합화 오행 논쟁 (아래 학파 이견) |

```
육합의 실전 적용:
- 육합은 두 지지가 결합하여 본래 기능이 약해지거나 변화하는 것
- 합거 판단: 합한 지지가 용신의 통근처이면 → 용신 약화 가능
- 합화 판단: 월령의 지지를 받아야 합화 성립 (천간합과 동일 원칙)
- 합이 충을 만나면: 충산합해(沖散合解) — 충이 합을 깨뜨림
```

### 2-2. 지지삼합(三合)

| 삼합 | 합화 오행 | 구성 |
|---|---|---|
| 寅午戌 | 火 | 생지(寅) + 왕지(午) + 묘지(戌) |
| 申子辰 | 水 | 생지(申) + 왕지(子) + 묘지(辰) |
| 亥卯未 | 木 | 생지(亥) + 왕지(卯) + 묘지(未) |
| 巳酉丑 | 金 | 생지(巳) + 왕지(酉) + 묘지(丑) |

```
삼합의 판단 규칙:
① 3자 완전 삼합: 세 글자가 모두 있으면 합화 성립
   → 세 지지의 원래 오행은 소멸하고 합화 오행으로 변함
   
② 반합(半合): 3자 중 2자만 있는 경우
   - 왕지 포함 반합: 합력이 강함 (예: 寅午, 午戌)
   - 왕지 미포함 반합: 합력이 약함 (예: 寅戌)
   → 반합은 합화까지 가지 않고 합의 경향만 있음

③ 대운·세운에서 삼합 완성:
   - 원국에 2자 + 대운·세운에 1자 → 해당 시기 삼합 완성
   - 합화 오행이 용신이면 길, 기신이면 흉

⚠️ 삼합에 묶인 지지가 일간의 통근처일 때:
   → 지지의 오행이 합화 오행으로 변질
   → 일간이 뿌리를 잃는 결과 초래 가능
   → 반드시 일간의 근기 유지 여부를 먼저 확인
```

### 2-3. 지지방합(方合)

| 방합 | 합화 오행 | 구성 |
|---|---|---|
| 寅卯辰 | 木 | 동방 목국 |
| 巳午未 | 火 | 남방 화국 |
| 申酉戌 | 金 | 서방 금국 |
| 亥子丑 | 水 | 북방 수국 |

```
방합의 특성:
- 삼합보다 합력이 강하다
- 같은 방위의 세 지지가 모이면 해당 오행이 극대화
- 실전에서 방합이 완전히 갖춰지는 사주는 드물지만,
  갖춰지면 해당 오행이 사주를 압도 → 종격 가능성 검토
```

## 3. 지지충(地支沖)

### 3-1. 6종 지지충

| 충 | 성격 | 영향 |
|---|---|---|
| 子午충 | 수화상충 | 감정·심장·비뇨기 관련 급변 |
| 丑未충 | 토토상충 | 재산·부동산·위장 관련 변동 |
| 寅申충 | 목금상충 | 직업·이동·간담·폐 관련 |
| 卯酉충 | 목금상충 | 문서·계약·간·호흡기 관련 |
| 辰戌충 | 토토상충 | 변화·이사·창고 관련 |
| 巳亥충 | 화수상충 | 이동·여행·심장·신장 관련 |

### 3-2. 충의 판단 규칙

```
기본 원칙:
- 충은 두 지지가 정면 대립하여 서로의 기능을 파괴하는 관계
- 힘이 강한 쪽이 약한 쪽을 이기고, 약한 쪽이 파괴됨
- 월령을 받은 쪽 > 받지 못한 쪽

충의 강도 판단:
① 왕충쇠(旺沖衰): 왕한 지지가 쇠한 지지를 충 → 쇠한 쪽 파괴
② 쇠충왕(衰沖旺): 쇠한 지지가 왕한 지지를 충 → 오히려 자기가 파괴
③ 균등충: 양쪽 힘이 비슷 → 양쪽 모두 불안정

실전 적용:
- 용신의 통근처가 충당하면 → 용신 약화 → 흉
- 기신의 통근처가 충당하면 → 기신 제거 → 길
- 월지가 충당하면 → 격국의 근거 동요 → 대사건

충산합해(沖散合解):
- 합과 충이 동시에 발생하면 충이 합을 깨뜨림
- 예) 子丑합 상태에서 午가 들어오면 → 子午충 → 子丑합 해체

원국 충 vs 대운·세운 충:
- 원국 내 충: 태어날 때부터의 구조적 긴장 → 평생 이 방면에서 변동
- 대운·세운 충: 특정 시기의 사건 → 해당 기간에 변동 발생
```

## 4. 지지형(地支刑)

### 4-1. 3종 지지형

```
① 무례지형(無禮之刑) — 상호 형
   寅巳형, 巳申형, 寅申형
   → 寅巳申 삼형: 세 글자가 모두 모이면 형의 힘이 극대화
   
② 무은지형(無恩之刑) — 상호 형
   丑戌형, 戌未형, 丑未형
   → 丑戌未 삼형: 토(土)끼리의 형 → 토기 과잉·혼탁
   
③ 자형(自刑)
   辰辰형, 午午형, 酉酉형, 亥亥형
   → 같은 지지가 2개 이상 → 자기모순·내부갈등
```

### 4-2. 형의 판단 규칙

```
형의 성격:
- 충(沖)이 정면 대립이라면, 형(刑)은 비틀어진 갈등
- 형은 법률·규율·질서와 관련된 문제로 나타나는 경향
- 관재(官災), 시비, 수술, 사고 등의 형태

형의 강도:
- 삼형(3자 모두 모임) > 이형(2자만 모임)
- 월지가 형에 관여 > 다른 지지가 관여
- 대운·세운에서 삼형이 완성되는 시기 → 관재·건강 주의

⚠️ 형의 해석 시 학파 이견 크다 (23번 문서 참조):
- 적천수파: 형의 영향을 크게 봄 — 관재·형벌의 상(象)
- 자평진전파: 형의 영향을 상대적으로 작게 봄 — 충보다 약한 작용
- 현대 명리: 형을 합충 다음 순위로, 중요하되 과대해석 경계
```

## 5. 지지파(地支破)

### 5-1. 6종 지지파

| 파 | 구성 |
|---|---|
| 子酉파 | 수금 — 금생수인데 파? (음양 불조화) |
| 丑辰파 | 토토 — 같은 오행의 파 |
| 寅亥파 | 목수 — 수생목인데 파? (합과 파의 공존) |
| 卯午파 | 목화 — 목생화인데 파? (음양 불조화) |
| 巳申파 | 화금 — 화극금의 변형 |
| 未戌파 | 토토 — 같은 오행의 파 |

### 5-2. 파의 판단 규칙

```
파의 성격:
- 파(破)는 합충형보다 영향이 약함
- "깨뜨림"이라는 뜻으로, 기존 관계나 계획이 틀어지는 상(象)
- 특히 합(合) 관계에 있는 지지의 대각선 관계가 파에 해당

실전 적용:
- 파는 단독으로 큰 사건을 만들지 않음
- 충이나 형과 겹칠 때 부가적으로 고려
- 대부분의 학파에서 합·충·형에 비해 비중을 낮게 둠

⚠️ 寅亥파의 특수성:
- 寅亥는 육합(合木)이면서 동시에 파(破)
- 합과 파가 공존 → "합하면서 깨지는" 복잡한 관계
- 표면적 협력 속에 내면적 갈등이 있는 구조
```

## 6. 지지해(地支害)

### 6-1. 6종 지지해

| 해 | 구성 | 유래 |
|---|---|---|
| 子未해 | 수토 | 子丑합을 午가 충하여 생기는 해 |
| 丑午해 | 토화 | 子丑합을 午未합이 방해 |
| 寅巳해 | 목화 | 寅亥합을 巳가 방해 |
| 卯辰해 | 목토 | 卯戌합을 辰이 방해 |
| 申亥해 | 금수 | 巳申합을 亥가 방해 |
| 酉戌해 | 금토 | 辰酉합을 戌이 방해 |

### 6-2. 해의 판단 규칙

```
해의 성격:
- 해(害)는 "穿"이라고도 부름 — 뚫는다는 뜻
- 합하려는 관계를 제3자가 방해하여 원망이 생기는 관계
- 인간관계의 배신, 시기, 모함과 연결

실전 비중:
- 합·충·형보다 한 단계 아래의 영향
- 단독으로 대사건을 만들지 않으나, 인간관계 갈등의 배경으로 작용
- 대운·세운에서 해가 겹칠 때 → 배신·관계 갈등 주의

⚠️ 해의 비중에 대한 학파 이견:
- 전통파: 해를 합충형과 함께 중요하게 봄
- 현대 실전파: 해의 비중을 낮추는 경향 — "충이 주, 해는 부"
```

## 7. 합충형파해 우선순위와 상호작용

```
강도 순위 (일반적):
천간합 ≥ 지지충 > 삼합·방합 > 지지형 > 육합 > 해 > 파

상호작용 규칙:
① 충산합해(沖散合解): 충이 합을 깨뜨림
   → 예) 卯戌합 + 辰戌충 → 충이 합을 깨고 戌을 흔듦

② 합동충(合同沖): 합과 충이 같은 글자에 동시 작용 → 충 우세

③ 형과 충의 동시 발생:
   → 寅申충 + 巳申형이 같은 대운에서 → 충의 파괴력에 형의 관재성 추가

④ 삼합 vs 충: 삼합 중 1자가 충당하면 삼합이 깨질 수 있음
   → 예) 寅午戌 삼합 중 子가 午를 충 → 삼합 와해 가능

⑤ 원국 합 + 대운 충: 대운의 충이 원국의 합을 깨뜨릴 수 있음
   → 합으로 안정되어 있던 구조가 대운 충으로 불안정해짐
```

## 8. 신살(神煞) — 주요 항목과 실전 비중

### 8-1. 신살 총론

```
신살의 위상:
- 고전 명리(연해자평 시대): 신살을 격국과 동등하게 중시
- 자평진전·적천수 이후: 신살의 비중을 대폭 낮춤
- 현대 실전: 격국·용신 분석이 우선, 신살은 참고·보조 수준

원칙:
"격국과 용신·기신 분석이 주(主)이고, 신살은 보(補)이다."
신살만으로 길흉을 단정하지 않으며, 격국 분석 결과와 일치할 때만 보강 근거로 사용한다.
```

### 8-2. 주요 길신(吉神)

```
천을귀인(天乙貴人):
- 일간 기준: 甲戊庚→丑未, 乙己→子申, 丙丁→亥酉, 壬癸→巳卯, 辛→午寅
- 의미: 위기에 귀인의 도움을 받는 상(象)
- 실전 비중: 중간 — 격국이 좋으면 귀인의 도움이 실현, 격국이 나쁘면 유명무실

문창귀인(文昌貴人):
- 일간 기준: 甲→巳, 乙→午, 丙→申, 丁→酉, 戊→申, 己→酉, 庚→亥, 辛→子, 壬→寅, 癸→卯
- 의미: 학문·문서·시험 운
- 실전 비중: 낮음~중간

천덕귀인(天德貴人)·월덕귀인(月德貴人):
- 월지 기준으로 결정
- 의미: 재앙을 면하는 보호의 상(象)
- 실전 비중: 낮음
```

### 8-3. 주요 흉살(凶煞)

```
양인(羊刃):
- 일간의 제왕지(帝旺支): 甲→卯, 丙戊→午, 庚→酉, 壬→子
  (음간은 양인이 없다는 견해가 주류)
- 의미: 극단적 강함 → 과격·사고·수술의 상(象)
- 실전 비중: 높음 — 특히 양인이 충당하거나 관살과 만날 때 사건 발생
- ⚠️ 양인은 신살 중 실전 비중이 가장 높다
  → 격국론에서도 양인격(양인합살)으로 별도 취급하는 학파 있음

역마(驛馬):
- 일지(또는 년지) 기준: 寅午戌→申, 申子辰→寅, 亥卯未→巳, 巳酉丑→亥
- 의미: 이동·변동·출장·해외의 상(象)
- 실전 비중: 중간 — 대운·세운에서 역마가 활성화될 때 이동 가능성

도화(桃花) / 함지(咸池):
- 일지(또는 년지) 기준: 寅午戌→卯, 申子辰→酉, 亥卯未→子, 巳酉丑→午
- 의미: 이성·매력·예술·유흥의 상(象)
- 실전 비중: 낮음~중간 — 과대해석 주의

공망(空亡):
- 일주 기준 60갑자 순(旬)에서 빠진 2개 지지
- 예) 甲子순(甲子~癸酉): 戌亥 공망
- 의미: 해당 지지의 기능이 비어 있음 → 뜻은 있으나 실현이 안 됨
- 실전 비중: 학파 이견 큼
  → 공망을 중시하는 학파: 공망 지지는 힘이 없다
  → 공망을 경시하는 학파: 대운에서 공망이 채워지면 정상 작동

화개(華蓋):
- 일지(또는 년지) 기준: 寅午戌→戌, 申子辰→辰, 亥卯未→未, 巳酉丑→丑
- 의미: 고독·종교·학문·예술적 기질
- 실전 비중: 낮음
```

### 8-4. 신살 적용 원칙

```
① 격국 분석이 주(主), 신살은 부(補)
   → 신살이 격국 분석과 모순되면 격국을 따름

② 신살만으로 길흉을 단정하지 않음
   → "도화살이 있으니 바람둥이" 식의 단정은 오류
   → 격국 구조상 식상이 왕한 사주에 도화가 있으면 → 보강 근거 정도

③ 양인·역마·공망은 상대적으로 비중이 높음
   → 이 세 가지는 격국 분석에도 영향을 줄 수 있는 신살
   → 나머지 신살은 참고 수준

④ 대운·세운에서 신살이 활성화될 때
   → 원국에 신살이 있고, 대운·세운에서 해당 지지가 충·합으로 활성화
   → 신살적 사건(이동, 이성, 관재 등)이 발생할 가능성
   → 단, 격국의 길흉 판단이 우선
```

## 9. 합충형파해 실전 체크리스트

```
사주 분석 시 합충형파해 확인 순서:

□ Step 1: 천간합 확인
  → 천간 4자 중 합이 있는가?
  → 합거되는 글자가 용신인가 기신인가?

□ Step 2: 지지충 확인
  → 지지 4자 중 충이 있는가?
  → 충당하는 지지가 월지(격국 근거)인가?
  → 충당하는 지지가 용신의 통근처인가?

□ Step 3: 삼합·방합 확인
  → 삼합이 완성되는가? 반합인가?
  → 합화 오행이 용신 계열인가 기신 계열인가?

□ Step 4: 육합 확인
  → 육합으로 묶이는 지지가 있는가?
  → 합으로 인해 지지의 원래 기능이 약해지는가?

□ Step 5: 형 확인
  → 삼형이 존재하는가?
  → 관재·건강 관련 형이 있는가?

□ Step 6: 파·해 확인 (보조)
  → 파·해가 있으면 참고 기록, 단독으로 대사건 판단 않음

□ Step 7: 대운·세운 합충 추가 확인
  → 대운·세운 글자가 원국과 새로운 합충을 일으키는가?
  → 원국의 기존 합을 깨뜨리는가 (충산합해)?
  → 원국의 기존 충에 가세하는가?
```
