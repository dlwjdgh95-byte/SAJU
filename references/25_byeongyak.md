# 병약론(病藥論) — 格보다 앞서는 최우선 판단 층위

> **읽는 시점**: Step 4 격국 판정 직후, 그리고 Step 7-3 대운 길흉 판단 전. SKILL.md의 병약론 요약 체크리스트에서 이 파일로 라우팅된다.
> 이 문서는 구 SKILL.md 본문(539~633행)의 전문을 그대로 옮긴 것이다. 라이브 교정으로 다듬어진 원문이므로 임의 수정하지 않는다.

> "격국의 공식을 적용하기 전에, 사주에 **病(병)**이 있는지 먼저 진단하라."
> 格의 용신·기신은 '정상적인 균형' 상태를 전제한다. 병이 든 사주에서 格의 공식을 기계적으로 적용하면 용신과 기신이 정반대로 뒤집힌다.

### 病의 판단 — 설기태심(洩氣太甚)과 오행의 편중

**病이란**: 사주에서 특정 오행이 지나치게 강해 일간이나 격국의 균형을 무너뜨리는 상태.

```
대표적인 病의 유형:

① 설기태심(洩氣太甚): 식상이 너무 많아 일간 에너지가 과도하게 빠져나감
   → 병 = 과다한 식상 / 약 = 인성(식상을 극제)

② 토다매화(土多埋火): 土기가 너무 많아 火일간이 묻혀버림
   → 병 = 과다한 土 / 약 = 木(목극토)

③ 재다신약(財多身弱): 재성이 너무 많아 일간이 감당 못함
   → 병 = 과다한 재성 / 약 = 비겁(일간 보강) 또는 인성

④ 살강무제(殺强無制): 관살이 너무 강하고 제어 수단이 없음
   → 병 = 강한 관살 / 약 = 식신(제살) 또는 인성(화살)
```

### 조토불생금(燥土不生金) — 계절·질(質)에 따른 오행 기능 판단

土가 있어도 **건조한 土(조토)**는 金을 생하지 못한다. 오행의 생극은 오행의 종류뿐 아니라 **계절과 질(質)**에 따라 달라진다.

```
조토(燥土): 未·戌 — 여름·가을 건조한 흙. 金을 생하는 능력 약함.
            金은 이 위에서 힘을 받지 못하고 열기에 녹아내림.
습토(濕土): 丑·辰 — 겨울·봄 촉촉한 흙. 金을 제대로 생함.

→ 조토 위의 金을 용신으로 보는 것은 오류.
  통근처가 조토뿐인 金은 실제 힘이 없다.
```

### 탐재괴인(貪財壞印) — 재성이 인성(용신)을 파괴하는 경우

인성이 진짜 용신일 때, 재성이 인성을 극(財克印)하면 → **탐재괴인**: 재성을 탐하다 인성(용신)을 망침.

```
판단 원칙:
- 인성이 용신인 사주에서 재성 대운은 기신이다.
- "재성 = 식신생재의 결실"이라는 공식에 집착하면
  탐재괴인을 見하고 대운을 정반대로 해석하게 된다.
```

### 상관패인(傷官佩印) — 식상 과다 시 인성이 귀격

식상이 과다하여 병이 든 사주에서, 인성이 식상을 제어하면 → **상관패인**: 자평진전이 인정하는 귀격.

```
정상적인 식신격: 식신 → 재성(용신), 편인(기신)
식상 과다·병든 사주: 인성(용신), 재성(기신/탐재괴인)
→ 같은 格이어도 사주의 병 여부에 따라 용신·기신이 완전히 뒤집힌다.
```

### 병약론 적용 프로세스

```
格 분석 후, 반드시 아래를 먼저 검토:

Step A: 이 사주에 '병'이 있는가?
        → 특정 오행이 3개 이상 편중? 일간 설기가 지지 과반을 차지?
        → YES → 병이 있는 사주 → 병약론 적용

Step B: 병(病)이 무엇인가?
        → 과다한 오행을 특정

Step C: 약(藥)이 무엇인가?
        → 병을 극제하는 오행 = 진짜 용신

Step D: 格의 공식 용신이 약(藥)과 같은가, 다른가?
        → 같으면: 格 공식 그대로 적용
        → 다르면: 병약론 우선 → 格 공식 수정
                  이때 格의 기신이 실제 약(藥)이 되는 역전 발생

Step E: 탐재괴인 여부 확인
        → 진짜 용신(인성 등)을 파괴하는 오행이 있는가?
        → 있으면: 해당 오행은 格의 공식과 무관하게 기신
```

### 주의: 병약론이 적용되지 않는 경우

병약론은 오행이 극단적으로 편중된 사주에만 적용한다. 균형 잡힌 사주에 무리하게 적용하면 오히려 格을 왜곡한다.

```
병약론 적용 기준:
- 특정 오행이 지지 4자 중 3개 이상 점유
- 또는 천간·지지 합산 8자 중 5개 이상 동일 오행 계열
- 일간의 설기량이 생조량의 2배 이상
→ 이 기준 미달 시: 格 공식 우선 적용
```
