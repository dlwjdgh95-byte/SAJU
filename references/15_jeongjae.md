# 자평진전(子平眞詮) — 정재격(正財格)

> 자평진전 격국론 시리즈 5편.

---

## 1. 정의와 본질

정재격(正財格)은 **월지 장간 중 정재(正財)가 천간에 투출**하여 성립한다.
정재(正財)란 일간이 음양이 다르게 극(剋)하는 오행이다 — 내가 다스리는 재물.

| 일간 | 정재 |
|---|---|
| 甲(양목) | 己(음토) |
| 乙(음목) | 戊(양토) |
| 丙(양화) | 辛(음금) |
| 丁(음화) | 庚(양금) |
| 戊(양토) | 癸(음수) |
| 己(음토) | 壬(양수) |
| 庚(양금) | 乙(음목) |
| 辛(음금) | 甲(양목) |
| 壬(양수) | 丁(음화) |
| 癸(음수) | 丙(양화) |

**정재의 본질**: 내가 통제하고 관리하는 안정적인 재물. 노력으로 획득하는 현실적 재물이다. 편재(偏財)가 유동적·투기적 재물이라면, 정재는 성실하게 쌓아가는 재물이다.

---

## 2. 성립 조건

월지 장간 중 정재가 천간에 투출하면 格이 성립한다.

**정재격의 전제**: 일간이 재를 통제할 수 있어야 한다. 일간이 너무 약하면 재가 오히려 일간을 압박한다(재다신약, 財多身弱).

---

## 3. 용신·희신·기신

### 용신

**① 식상(食傷) — 식상생재(食傷生財)**

식상이 재성을 생하면 재물이 더욱 풍성해진다. 내 능력(식상)으로 재물(재성)을 만드는 형상이다.

```
구조: 일간 →설기→ 식상(食傷) →생→ 정재(財)
```

**② 관성(官星) — 재생관(財生官)**

정재가 관성을 생하는 구조. 재물이 명예(관)를 만드는 형상이다. 정재격에서 관성이 있으면 재관쌍미(財官雙美)라 하여 귀하다.

### 기신

**① 겁재(劫財)·비견(比肩) — 겁재탈재(劫財奪財)**

> "겁재가 재성을 탈취한다"

겁재·비견이 정재를 극(比劫剋財)하면 정재격이 가장 심각하게 손상된다. 내 재물을 타인이 빼앗아가는 형상이다.

**② 식상 없는 신약(身弱)**

일간이 약하고 식상도 없는데 정재만 강하면, 재가 일간을 역극(逆剋)하는 재다신약(財多身弱) 상황이 된다.

---

## 4. 성격(成格)

### 유형 ① 식상생재(食傷生財)

```
예시: 甲木 일간, 未월(己土 정재 투출), 천간에 丙火 식신
      甲木→丙火(식신)→己土(정재) → 식상생재 완성
```

내 능력으로 재물을 만드는 가장 이상적인 정재격 구조.

### 유형 ② 재관쌍미(財官雙美)

```
예시: 甲木 일간, 未월(己土 정재), 천간에 辛金 정관
      己土가 辛金을 생(土生金) → 재관쌍미
      재물로 명예를 이루는 형상
```

---

## 5. 패격(敗格)

### 유형 ① 겁재탈재(劫財奪財)

```
예시: 甲木 일간, 未월(己土 정재), 천간에 乙木 겁재
      乙木이 己土를 극(木克土) → 재성 파괴 → 패격
```

### 유형 ② 재다신약(財多身弱)

```
예시: 甲木 일간이 약한데 재성이 너무 많음
      재가 일간을 역압 → 재물이 있어도 감당 못하는 형상
```

---

## 6. 구격(救格)

### 유형 ① 관성제겁(官星制劫)

겁재가 재성을 탈취할 때, 관성이 겁재를 극(官克劫)하여 재성을 보호한다.

```
예시: 乙木 겁재(기신)가 있을 때 辛金 정관이 乙木을 극(金克木)
      → 겁재 제어 → 재성 보호 → 구격
```

### 유형 ② 인성생신(印星生身)

재다신약 상황에서 인성이 일간을 생하여 일간을 강하게 만들어 재를 감당하게 한다.

---

## 7. 정재격의 특수 구조

**재용식생(財用食生)**: 식상이 재를 생하는 구조. 실업가·사업가의 명.

**재용관성(財用官星)**: 재가 관을 생하는 구조. 재물로 명예를 얻는 형상.

**재용인수(財用印綬)**: 재다신약일 때 인성으로 일간을 보강하는 구조. 역설적이지만 이 경우 인성이 용신이 된다.

---

## 8. 학파 간 이견

**이견 ①** 겁재탈재의 절대성: 자평진전은 겁재가 있으면 반드시 기신이라 하지만, 겁재와 재성의 세력 비교 및 관성의 존재 여부에 따라 판단해야 한다는 견해가 있다.

**이견 ②** 재다신약의 기준: 재다신약의 기준이 학파마다 다르다. 일부는 신강·신약보다 격국의 성패를 더 중시한다.

---

## 9. 정재격 핵심 요약

성격: 식상생재(내 능력으로 재물 생산) 또는 재관쌍미
패격: 겁재탈재(겁재가 재성 탈취) 또는 재다신약
구격: 관성이 겁재를 제어하여 재성 보호
