# 자평진전(子平眞詮) — 격국 사례 분석

> 이 문서는 격국론의 이론을 실제 사주 구조에 적용한 사례들을 정리한 것이다.
> 원전 수록 사례와 현대 명리학의 대표 사례를 함께 담았다.

---

## 1. 정관격 사례

### 사례 1-A: 정관격 성격 (관인상생)

```
사주 구조:
年: 甲子
月: 辛酉
日: 甲午 (甲木 일간)
時: 壬子

분석:
月支 酉中 辛金이 月干에 투출 → 甲木 일간의 정관격 성립
壬水 정인이 時干에 투출, 子水에 통근 → 인성 힘 있음
辛金(정관) → 壬水(정인) → 甲木(일간): 관인상생 구조 완성
기신(상관·겁재·칠살) 없음

판정: 성격 — 관인상생 구조의 정관격
특징: 학문과 명예의 명. 관직·교육계에서 두각을 나타내는 구조.
```

### 사례 1-B: 정관격 패격 (상관견관)

```
사주 구조:
年: 壬午
月: 辛酉
日: 甲辰 (甲木 일간)
時: 丁卯

분석:
月支 酉中 辛金 정관 투출 → 정관격 성립
時干 丁火 상관 투출, 卯木에 통근 → 상관 힘 있음
丁火(상관)가 辛金(정관)을 극(火克金) → 상관견관 발생

기신 제어 여부 확인:
壬水(年干)가 있으나 丁壬합으로 壬水가 합거됨
→ 壬水의 정인 역할 소멸 → 상관을 제어할 수단 없음

판정: 패격 — 상관견관이 해소되지 않음
특징: 관직에서 충돌이 많고 직업 변동이 잦은 형상.
```

### 사례 1-C: 정관격 구격 (상관패인)

```
사주 구조:
年: 庚申
月: 辛酉
日: 甲午 (甲木 일간)
時: 丁丑

분석:
辛金 정관 투출 → 정관격 성립
丁火 상관 투출 → 상관견관 발생 → 패격 위기
癸水(丑中 장간) 또는 壬水 인성이 있어야 구격 가능

추가 조건: 만약 年干에 壬水가 있다면
壬水(정인)가 丁火(상관)를 극(水克火) → 상관 제어
→ 상관패인 구조 완성 → 구격 → 패중취귀

판정: 구격 조건부 — 인성의 존재 여부에 따라 달라짐
```

## 2. 칠살격 사례

### 사례 2-A: 칠살격 성격 (식신제살)

```
사주 구조:
年: 丙午
月: 庚申
日: 甲申 (甲木 일간)
時: 丙午

분석:
月支 申中 庚金이 月干에 투출 → 甲木 일간의 칠살격 성립
庚金(칠살) 강함 — 月日 모두 申, 年月時에 庚·丙
丙火(식신) 투출, 午火에 통근 → 식신 힘 있음
丙火(식신)가 庚金(칠살)을 극(火克金) → 식신제살 완성
재성 없어 재성생살 위험 없음

판정: 성격 — 식신제살 구조의 칠살격
특징: 강한 살을 식신이 완전히 제어하여 권위 있는 명.
```

### 사례 2-B: 칠살격 패격 (재성생살)

```
사주 구조:
年: 戊戌
月: 庚申
日: 甲子 (甲木 일간)
時: 戊午

분석:
庚金 칠살 투출 → 칠살격 성립
戊土(편재)가 年·時干에 투출, 戌·午에 통근 → 재성 강함
戊土(편재)가 庚金(칠살)을 생(土生金) → 재성생살 발생
칠살이 더욱 강해짐 → 甲木 일간 압박 심화
식신이 없어 살을 제어할 수단 없음

판정: 패격 — 재성생살이 해소되지 않음
특징: 재물이 오히려 화를 불러오는 형상. 사업 실패의 패턴.
```

## 3. 정인격 사례

### 사례 3-A: 정인격 성격 (관인상생)

```
사주 구조:
年: 己丑
月: 癸亥
日: 壬子 (壬水 일간)
時: 辛卯

분석:
月支 亥中 壬水가 아닌 甲木... 

재확인: 壬水 일간의 정인은 辛金
時干 辛金(정인)이 卯木에 뿌리가 없지만...
月干 癸水는 겁재

더 좋은 예시로 재구성:
壬水 일간, 酉월(辛金 정인 투출), 己土(정관) 있음
己土(정관)가 辛金(정인)을 생(土生金) → 辛金이 壬水를 생(金生水)
→ 관인상생 완성

판정: 성격 — 관인상생 구조의 정인격
특징: 학문으로 관직에 오르는 구조.
```

### 사례 3-B: 정인격 패격 (재파인)

```
사주 구조:
壬水 일간, 酉월(辛金 정인 투출)
天干: 甲木 식신, 丙火 편재

분석:
辛金(정인) 투출 → 정인격 성립
丙火(편재)가 辛金(정인)을 극(火克金) → 재파인 발생
甲木(식신)이 丙火를 생하여 재성을 더욱 강하게 함
정인이 재성에 의해 파괴됨

판정: 패격 — 재파인 해소 안 됨
구격 조건: 비겁(壬水 동류 또는 癸水)이 있어 丙火 재성을 제어하면 구격 가능
```

## 4. 식신격 사례

### 사례 4-A: 식신격 성격 (식신생재)

```
사주 구조:
甲木 일간, 午월(丙火 식신 투출)
天干: 戊土 편재 있음
地支: 뿌리 있음

분석:
丙火(식신) 투출, 午火에 통근 → 식신 힘 있음
戊土(편재)가 있음, 지지에 뿌리
丙火(식신)가 戊土(편재)를 생(火生土) → 식신생재 완성
편인(壬水) 없음 → 효신도식 위험 없음

판정: 성격 — 식신생재 구조
특징: 표현 능력으로 재물을 만드는 복록(福祿)의 명.
```

### 사례 4-B: 식신격 패격 (효신도식)

```
사주 구조:
甲木 일간, 午월(丙火 식신 투출)
天干: 壬水 편인 있음

분석:
丙火(식신) 투출 → 식신격 성립
壬水(편인)가 丙火(식신)를 극(水克火) → 효신도식 발생
식신의 생산·복록 기능 박탈

구격 조건:
戊土(재성)가 있으면 戊土가 壬水(편인)를 극(土克水)
→ 편인 제어 → 식신 보호 → 구격

판정: 패격 (구격 조건 없을 때)
특징: 노력해도 결실이 사라지는 형상. 의식주 불안정.
```

## 5. 상관격 사례

### 사례 5-A: 상관격 성격 (상관생재)

```
사주 구조:
庚金 일간, 子월(癸水 상관 투출)
天干: 甲木 편재 있음
관성 없음

분석:
癸水(상관) 투출, 子水에 통근 → 상관격 성립
甲木(편재)가 있음, 지지에 뿌리
癸水(상관)가 甲木(편재)를 생(水生木) → 상관생재 완성
정관 없어 상관견관 위험 없음

판정: 성격 — 상관생재 구조
특징: 날카로운 능력으로 재물을 만드는 형상.
```

### 사례 5-B: 금수상관 성격 (상관격 + 관성)

```
사주 구조:
庚金 일간, 子월(癸水 상관 투출)
天干: 己土 정관 있음

분석:
癸水(상관) 투출 → 상관격 성립
己土(정관)가 있음

일반 상관격: 상관견관으로 패격
그러나 금수상관(庚金 일간, 水 상관) 특례:
"금수상관희견관(金水傷官喜見官)"
→ 금수 상관격에서 관성은 오히려 격을 완성

판정: 성격 — 금수상관의 특례 적용
특징: 금수 상관의 독특한 구조. 지성과 권위가 결합된 명.
```

### 사례 5-C: 상관격 패격 (상관견관)

```
사주 구조:
甲木 일간, 午월(丁火 상관 투출)
天干: 辛金 정관 있음
인성 없음

분석:
丁火(상관) 투출 → 상관격 성립
辛金(정관)이 있음
丁火(상관)가 辛金(정관)을 극(火克金) → 상관견관 발생
인성이 없어 상관을 제어할 수단 없음

판정: 패격 — 상관견관 해소 불가
구격 조건: 壬水·癸水 인성이 있으면 인성이 상관 제어 → 상관패인 → 구격
```

## 6. 복합 사례 — 패중취귀(敗中取貴)

### 사례 6-A: 정관격 패중취귀

```
사주 구조:
辛金 일간, 卯월(乙木 정관 투출)
天干: 丙火 상관, 壬水 정인, 戊土 편재

분석 순서:

① 格 결정:
   卯中 乙木이 年干에 투출 → 辛金 일간의 정관격 성립

② 기신 확인:
   丙火(상관)가 乙木(정관)을 극(火克金... 아니라 火는 辛金을 극)
   
   재확인: 辛金 일간의 정관은 丙火
   → 月支 卯中 乙木은 辛金의 편관(칠살)
   
   다시 설정: 辛金 일간, 午월(丙火 정관 투출)
   天干: 壬水 상관, 戊土 인성(편인), 甲木 재성

③ 재분석:
   丙火(정관) 투출 → 정관격
   壬水(상관)가 丙火(정관)를 극(水克火) → 상관견관 → 패격 위기
   戊土(편인)가 壬水(상관)를 극(土克水) → 상관 제어 → 구격
   甲木(재성)이 丙火(정관)를 생(木生火) → 재생관 → 격 보강

④ 판정: 패중취귀
   상관이 정관을 극하는 패격 원인이 있으나
   인성이 상관을 제어하고 재성이 정관을 생하는 구격이 이루어짐
   → 관·상관·인성·재성의 복합 구조 → 오히려 귀한 명
```

## 7. 格局 판단 Quick Reference

### 월지별 격국 성립 가능성

| 월지 | 주요 장간 | 가능한 格 (일간에 따라 다름) |
|---|---|---|
| 寅 | 甲丙戊 | 비겁·식신·편재격 가능 |
| 卯 | 甲乙 | 비겁격 가능 (격이 약함) |
| 辰 | 戊乙癸 | 편재·비겁·편인격 가능 |
| 巳 | 丙庚戊 | 식신·편관·편재격 가능 |
| 午 | 丁己 | 상관·편재격 가능 |
| 未 | 己丁乙 | 정재·상관·비겁격 가능 |
| 申 | 庚壬戊 | 편관·편인·편재격 가능 |
| 酉 | 辛庚 | 정관·편관격 가능 |
| 戌 | 戊辛丁 | 편재·정관·상관격 가능 |
| 亥 | 壬甲 | 편인·비겁격 가능 |
| 子 | 癸壬 | 상관·편인격 가능 |
| 丑 | 己癸辛 | 정재·상관·정관격 가능 |

※ 실제 格은 일간과의 관계로 결정됨. 위 표는 가능성의 범위만 제시.

### 格局별 핵심 판단 요약

```
정관격: 상관 있는가? (있으면 패격 위험) → 인성이 상관 제어하는가?
칠살격: 식신 있는가? (없으면 패격 위험) → 재성이 살 생하지 않는가?
정인격: 재성 있는가? (있으면 패격 위험) → 비겁이 재성 제어하는가?
편인격: 식신 있는가? (있으면 효신도식 위험) → 재성이 편인 제어하는가?
정재격: 겁재 있는가? (있으면 탈재 위험) → 관성이 겁재 제어하는가?
편재격: 겁재 있는가? + 칠살 있는가? → 각각 제어하는가?
식신격: 편인 있는가? (있으면 효신도식 위험) → 재성이 편인 제어하는가?
상관격: 정관 있는가? (있으면 상관견관) → 인성이 상관 제어하는가?
         금수상관이면 정관이 있어도 OK
```

## 8. 자평진전 원전의 핵심 명구(名句)

자평진전에서 격국론의 핵심을 담은 주요 표현들:

> "월령은 사주의 강령이다 (月令乃提綱之最重)"

> "정관은 하나면 귀하고 많으면 탁해진다 (官星一位最爲奇, 混雜相逢便不宜)"

> "상관견관, 화기백단 (傷官見官, 禍起百端)"

> "살이 있으면 반드시 제어해야 귀하다 (殺有制則貴, 無制則凶)"

> "식신제살, 위권최고 (食神制殺, 威權最高)"

> "재파인격, 화환무궁 (財破印格, 禍患無窮)"

> "금수상관희견관 (金水傷官喜見官)"

> "패중취귀, 구신이 있으면 귀해진다"
