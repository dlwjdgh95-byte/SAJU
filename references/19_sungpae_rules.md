# 자평진전(子平眞詮) — 성격·패격·구격 판단 규칙

> 이 문서는 8개 격국에 공통 적용되는 성패 판단 원칙을 정리한 것이다.
> 각 격국 문서와 함께 읽으면 실전 사주 판단에 직접 활용할 수 있다.

---

## 1. 성격(成格) 판단의 3가지 기준

### 기준 ①: 월령(月令) 득기(得氣)

격국의 핵심 오행(정관·칠살·인성·재성·식신·상관 등)이 월령에서 힘을 얻어야 한다.

```
판단 방법:
1. 월지 장간을 확인
2. 그 장간이 천간에 투출했는지 확인
3. 투출된 오행이 지지에 뿌리(통근)를 두고 있는지 확인
→ 세 조건이 갖춰질수록 격이 높음
```

### 기준 ②: 용신(用神)의 건재

각 格에서 요구하는 용신이 힘 있게 존재해야 한다.

| 격국 | 핵심 용신 |
|---|---|
| 정관격 | 재성(재생관) 또는 인성(관인상생) |
| 칠살격 | 식신(제살) 또는 인성(화살) |
| 정인격 | 관성(관인상생) |
| 편인격 | 관성 또는 재성(재제편인) |
| 정재격 | 식상(식상생재) 또는 관성(재생관) |
| 편재격 | 식상(식상생재) 또는 관성(재생관) |
| 식신격 | 재성(식신생재) 또는 칠살(식신제살) |
| 상관격 | 재성(상관생재) 또는 인성(상관패인) |

### 기준 ③: 기신(忌神)의 부재 또는 제어

각 格을 파괴하는 기신이 없거나, 있더라도 완전히 제어되어야 한다.

```
제어의 조건:
① 기신이 천간에 없는 경우 (지지에만 있으면 영향 감소)
② 기신이 다른 오행에 의해 극(剋)당하는 경우
③ 기신이 합거(合去)되어 기능을 잃은 경우
④ 기신의 세력이 용신보다 현저히 약한 경우
```

## 2. 格을 파(破)하는 주요 조합

### 공통 파격 조합

**① 관살혼잡(官殺混雜)**

정관과 칠살이 천간에 함께 나타나는 상황.
- 영향 격: 정관격, 칠살격 모두
- 해결: 합살유관(칠살이 합거) 또는 합관유살(정관이 합거)

**② 재다신약(財多身弱)**

재성이 너무 많고 일간이 약한 상황.
- 영향 격: 정재격, 편재격
- 일간의 힘이 재를 감당하지 못해 격이 손상됨

**③ 인성피재(印星被財)**

인성이 재성에 의해 극파되는 상황 (재파인).
- 영향 격: 정인격, 편인격, 상관패인 구조
- 해결: 비겁이 재성 제어

**④ 식신피편인극(食神被偏印剋)**

편인이 식신을 극하는 효신도식(梟神盜食).
- 영향 격: 식신격, 편인격
- 해결: 재성이 편인 제어

### 格별 핵심 파격 조합

| 격국 | 핵심 파격 조합 | 파격 원인 |
|---|---|---|
| 정관격 | 상관견관 | 상관이 정관 극파 |
| 정관격 | 겁재탈재 | 겁재가 재성(용신) 탈취 |
| 정관격 | 관살혼잡 | 칠살이 정관의 순수성 훼손 |
| 칠살격 | 재성생살 | 재가 살을 생해 살이 더 강해짐 |
| 칠살격 | 살강식약 | 살이 강하고 식신이 약해 제어 불가 |
| 정인격 | 재파인(財破印) | 재성이 인성을 직접 극파 |
| 편인격 | 효신도식 | 편인이 식신의 기능 박탈 |
| 정재격 | 겁재탈재 | 겁재가 정재를 탈취 |
| 편재격 | 겁재탈재 | 겁재가 편재를 탈취 |
| 편재격 | 재성생살 | 편재가 칠살을 생함 |
| 식신격 | 효신도식 | 편인이 식신 극파 |
| 식신격 | 살강식약 | 살이 강해 식신의 제살 기능 무력화 |
| 상관격 | 상관견관 | 상관이 정관을 극(충돌) |
| 상관격 | 재파인 | 재성이 인성(용신) 극파 |

## 3. 格을 구(救)하는 주요 조합

### 공통 구격 원리

**원리 ①: 기신을 극(剋)하는 오행 존재**

기신을 직접 극하는 오행이 사주에 있으면 기신의 기능이 약화된다.

```
기신 → 기신을 극하는 오행(구신) → 구격
```

**원리 ②: 기신을 합거(合去)하는 천간합**

기신이 다른 오행과 천간합을 이루어 본래 기능을 잃을 때 구격이 된다.

```
예: 상관(기신) + 다른 오행 → 합거 → 상관의 정관 극 기능 약화
```

**원리 ③: 용신을 생(生)하는 오행 존재**

용신이 약할 때 용신을 생하는 오행(희신)이 있으면 용신이 강화된다.

### 格별 핵심 구격 조합

| 격국 | 패격 원인 | 구격 방법 |
|---|---|---|
| 정관격(상관견관) | 상관이 정관 극 | 인성이 상관 극제(상관패인) |
| 정관격(겁재탈재) | 겁재가 재성 탈취 | 관성이 겁재 극제 |
| 정관격(관살혼잡) | 칠살 혼재 | 합살유관(칠살 합거) |
| 칠살격(재성생살) | 재가 살 생함 | 식신이 살 제어 후 재성도 제어 |
| 정인격(재파인) | 재가 인 극파 | 비겁이 재성 극제 |
| 편인격(효신도식) | 편인이 식신 극 | 재성이 편인 극제(재제편인) |
| 재격(겁재탈재) | 겁재가 재 탈취 | 관성이 겁재 극제 |
| 식신격(효신도식) | 편인이 식신 극 | 재성이 편인 극제 |
| 상관격(상관견관) | 상관이 정관 극 | 인성이 상관 극제(상관패인) |
| 상관격(재파인) | 재가 인 극파 | 비겁이 재성 극제 |

## 4. 패중취귀(敗中取貴) — 패격이 귀해지는 원리

> "패격이라도 구신이 있으면 귀해질 수 있다"

자평진전의 가장 중요한 통찰 중 하나다. 사주가 표면적으로 패격처럼 보여도, 구신(救神)이 기신을 완전히 제어하면 오히려 더 복잡하고 강력한 명이 된다.

```
패중취귀의 대표 구조:

[정관격 패중취귀]
정관격 → 상관이 정관 극(패격 발생) → 인성이 상관 제어(구격)
→ 관·상관·인성의 삼중 구조로 오히려 복잡하고 귀한 명

[칠살격 패중취귀]
칠살격 → 재성이 살 생(패격 발생) → 식신이 살 제어하고 재성도 간접 제어
→ 살·재·식신의 역동적 구조로 귀한 명
```

**패중취귀의 조건**:
1. 패격의 원인(기신)이 명확할 것
2. 구신이 기신을 완전히 제어할 것 (부분 제어는 불충분)
3. 구신 자체가 충분한 힘(통근)을 가질 것

## 5. 격국의 고저(高低) 판단 종합 기준

### 높은 격국 (귀명)

1. 격국의 핵심 오행이 월령 득기하고 천간 투출
2. 용신이 힘이 있고 지지에 뿌리가 있음
3. 기신이 전혀 없거나 완전히 제어됨
4. 천간이 맑고 오행이 단순하며 혼탁하지 않음
5. 용신·희신·기신의 관계가 명확함

### 낮은 격국

1. 격국 오행이 월령 득기했으나 천간 투출 없음
2. 용신이 있으나 뿌리가 약함
3. 기신이 있으나 부분적으로만 제어됨
4. 천간이 혼탁하여 오행이 복잡하게 얽힘

### 극히 낮은 격국 (천한 명)

1. 기신이 있고 구신도 없어 패격이 방치됨
2. 용신이 없거나 완전히 파괴됨
3. 관살혼잡·재파인·효신도식 등이 해결되지 않음

## 6. 천간 투출과 지지 통근의 중요성

### 투출(透出)이란

월지 장간이 천간에 드러나는 것. 투출이 있으면 격국의 힘이 강하고 명확해진다.

### 통근(通根)이란

천간에 있는 오행이 지지에 같은 오행(또는 같은 오행을 포함하는 지지)이 있어 힘을 얻는 것.

```
예: 천간의 甲木이 지지에 寅·卯·亥(甲木의 통근지)가 있으면 통근됨
```

통근 없는 천간은 뿌리 없는 나무 — 겉으로 있어 보이지만 실력이 없다.

### 투출+통근의 이상적 조합

격국에서 가장 강력한 구조:
1. 격국 오행이 월지에서 득기(得氣)
2. 천간에 투출
3. 천간의 해당 오행이 다른 지지에도 통근
→ 이 세 조건이 모두 갖춰지면 격이 강고(强固)하다

## 7. 혼탁(混濁)과 맑음(淸純) — 격국의 질적 차이

자평진전이 강조하는 중요 개념이다.

### 맑은(淸純) 사주

오행의 관계가 단순하고 명확하다. 格의 용신·기신이 뚜렷하며, 불필요한 오행이 없다.

### 혼탁(混濁)한 사주

여러 오행이 복잡하게 얽혀 格의 방향이 불분명하다. 용신과 기신이 서로 뒤섞여 格의 순수성이 손상된다.

```
혼탁의 대표 사례:
- 관살혼잡: 정관과 칠살이 함께 있어 관의 성격이 불분명
- 재인상충(財印相充): 재성과 인성이 서로 극하여 용신이 손상
- 식상혼잡: 식신과 상관이 함께 있어 설기의 방향이 분산
```

## 8. 학파 간 주요 이견 정리

### 이견 ①: 격국론 vs 억부론

**자평진전(격국론)**: 용신은 격국에서 취한다. 월령이 제강이다.

**적천수·현대 억부론**: 일간의 강약을 먼저 판단하고 용신을 취한다. 신강이면 극설, 신약이면 생조.

**정리**: 자평진전은 억부를 완전히 부정하지 않지만, 격국을 먼저 판단하고 그 안에서 신강·신약을 고려한다. 억부론은 格보다 일간 강약을 우선시한다.

### 이견 ②: 격국론 vs 조후론

**자평진전(격국론)**: 계절의 기후는 참고하지만, 격국 성패가 더 중요하다.

**궁통보감(조후론)**: 일간이 태어난 월의 기후와 필요한 오행을 용신으로 취한다.

**정리**: 현대 명리학은 두 이론을 병용하는 경향이 있다. 격국으로 기본 구조를 보고, 조후로 세부 보정을 한다.

### 이견 ③: 격국의 변화 — 종격(從格) 문제

**자평진전**: 팔격 중심이며 종격에 대한 논의가 상대적으로 적다.

**적천수 및 일부 학파**: 일간이 극도로 약하거나 강할 때 종격(從財格·從殺格·從強格 등)으로 판단한다.

**정리**: 종격의 성립 조건과 범위에 대한 학파 간 이견이 매우 크다.

### 이견 ④: 용신의 수

**자평진전**: 용신은 하나로 명확해야 한다. 여럿이면 혼탁해진다.

**현대 일부 학파**: 주용신(主用神)·보조용신의 개념으로 복수 용신을 인정한다.

## 9. 실전 판단 프로세스

자평진전 격국론을 실전에 적용할 때의 순서:

```
Step 1: 월지 확인
        → 월지의 장간을 확인

Step 2: 투출 확인
        → 어떤 장간이 천간에 투출했는지 확인
        → 투출된 오행으로 格 결정

Step 3: 格 확인
        → 투출된 오행이 일간과 어떤 관계인지 확인
        → 정관·칠살·인성·편인·재성·식신·상관 중 어느 格인지 결정

Step 4: 용신 결정
        → 해당 格에서 요구하는 용신이 사주에 있는지 확인
        → 용신의 강약(통근 여부) 확인

Step 5: 기신 확인
        → 格을 파괴하는 기신이 있는지 확인

Step 6: 성격·패격·구격 판단
        → 기신이 없거나 제어됨 → 성격
        → 기신이 있고 제어 안 됨 → 패격
        → 기신이 있지만 구신이 제어 → 구격(패중취귀 가능)

Step 7: 격국의 고저 판단
        → 용신의 강약 + 기신의 유무로 높고 낮음 결정
```
