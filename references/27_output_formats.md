# 출력 형식 템플릿 모음

> **읽는 시점**: 각 Step의 최종 출력 직전. 본문 판단이 끝난 뒤 형식만 참조한다.

## Step 1-5 대운 출력 형식
```
## 대운
연간: [음/양간] / 성별: [남/여] → [순행/역행]   ← 대운 순역 기준은 연간(年干). 일간 아님 (errata #1)
첫 대운 시작: [N세] ([출생연도 + N]년~)

| 대운 | 간지 | 나이 | 연도 | 주요 십신 |
|---|---|---|---|---|
| 1 | XX | N세~ | YYYY~ | [천간십신]·[지지십신] |
| 2 | XX | N+10세~ | ... | ... |
...
```

## Step 6 종합 해석 출력
다음 구조로 출력한다:

```
## 사주팔자
년주 | 월주 | 일주 | 시주
천간  |  천간 | 천간  | 천간
지지  |  지지 | 지지  | 지지

## 기본 정보
- 일간: [일간] ([오행] [음양])
- 월령: [월지] ([계절])
- 일간의 특성: [간략 설명]

## 십신 배치
[각 기둥의 천간·지지와 일간 대비 십신]

## 주요 합충
[해당 합충 목록]

## 격국 분석
- 格: [格 이름]
- 성립 근거: [월지 장간 → 투출 → 格]
- 용신: [용신과 근거]
- 기신: [기신과 근거]
- 판정: 성격/패격/구격 (근거 포함)

## 조후 분석
- 출생 월 조후 용신: [오행]
- 격국 용신과의 일치 여부: [일치/불일치/보완]
- 종합 용신: [최종 판단]

## 격국의 고저
[높은 격/낮은 격 판단과 근거]

## 병약론·억부론 검토
- 특정 오행 편중 여부: [지지 4자 중 3개 이상 동일 오행?]
- 병(病) 여부: [있음/없음 — 병이 있으면 약(藥)을 용신으로]
- 조토/습토 구분: [용신 통근처의 土가 조토인가 습토인가]
- 탐재괴인 여부: [재성이 인성(용신)을 극하는가]
- 일간의 통근처: [개수와 강도]
- 신강/신약 여부: [판단]
- 설기태심 여부: [식상의 설기량 vs 일간 뿌리·생조량 비교]
  → 설기태심이면 格의 공식을 역전하여 인성을 용신으로

## 종합 해석
[전체적 사주 구조의 특성과 의미]
```

## Step 7-6 대운 해석 출력
```
## 대운 해석

### [N세~ / 간지] — [종합 판정]
- 천간 [간]: [십신] — [용신/기신/중립] → [길/흉/중] (근거)
- 지지 [지]: [십신] — [용신/기신/중립] → [길/흉/중] (근거)
- 합충: [해당 합충과 그 영향]
- 종합: [선길후흉 등 판정] / [이 시기의 핵심 특성 1~2줄]
```

## Step 8-5 세운 출력
```
## 세운 분석

### [연도] [간지]년 — [종합 판정]
- 천간 [간]: [십신] — [원국/대운과의 관계] → [길/흉/중]
- 지지 [지]: [십신] — [원국/대운과의 관계] → [길/흉/중]
- 주요 합충:
  · [세운 글자] + [원국 글자]: [합충 종류] → [영향]
  · [세운 글자] + [대운 글자]: [합충 종류] → [영향]
- 대운과의 관계: [강화/역전/중립]
- 종합: [판정] / [해당 연도의 핵심 특성 1~2줄]
```

## Step 9-4 월운 출력
```
## 월운 분석

### [월] [간지]월 — [종합 판정]
- 천간 [간]: [십신] → [길/흉/중]
- 지지 [지]: [십신] → [길/흉/중]
- 천간·지지 질적 상호작용: [혼탁 현상 여부 — 예: 壬辰의 탁수 현상]
- 주요 합충: [원국/대운/세운과의 합충 및 영향]
- 월초·중순 vs 월말 전략: [천간 기운이 강한 시기 vs 지지 기운이 강해지는 시기]
- 세운 기조와의 관계: [강화/완화/역행]
- 종합: [판정] / [해당 월의 핵심 특성]
```
