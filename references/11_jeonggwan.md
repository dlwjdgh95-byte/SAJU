# 자평진전(子平眞詮) — 정관격(正官格)

> 자평진전 격국론 시리즈 1편. 팔격(八格) 중 가장 기본이 되는 격이다.

---

## 1. 정의와 본질

정관격은 **월지 장간 중 정관이 천간에 투출**하거나 월지 본기가 정관일 때 성립한다.
정관(正官)이란 일간을 음양이 다르게 극(剋)하는 오행이다.

| 일간 | 정관 |
|---|---|
| 甲(양목) | 辛(음금) |
| 乙(음목) | 庚(양금) |
| 丙(양화) | 癸(음수) |
| 丁(음화) | 壬(양수) |
| 戊(양토) | 乙(음목) |
| 己(음토) | 甲(양목) |
| 庚(양금) | 丁(음화) |
| 辛(음금) | 丙(양화) |
| 壬(양수) | 己(음토) |
| 癸(음수) | 戊(양토) |

**정관의 본질**: 일간을 법도 안에서 제어하는 존재. 강압적인 칠살(偏官)과 달리 정관은 예의와 규범 안에서 일간을 다스려 귀(貴)의 상징이 된다.

---

## 2. 성립 조건

월지 장간 중 정관이 천간에 투출하면 격이 성립한다. 월지 본기가 정관이면 투출 없이도 격이 성립하나 힘이 다소 약하다.

**핵심 원칙 — 정관의 순수성**

> "정관은 하나면 족하다. 많으면 탁해지고, 칠살이 섞이면 관살혼잡이 된다."

정관이 천간에 하나만 있어야 귀하다. 칠살(편관)과 함께 있으면 관살혼잡(官殺混雜)으로 격이 손상된다.

---

## 3. 용신·희신·기신

### 용신

**① 재성(財星) — 재생관(財生官)**

재성이 정관을 생하면 관의 기운이 강해진다. 재성 → 정관 → 일간의 흐름이 막힘 없이 이루어지는 이상적 구조다.

**② 인성(印星) — 관인상생(官印相生)**

정관이 인성을 생하고, 인성이 일간을 생하는 구조. 정관의 에너지가 인성을 거쳐 일간에 전달되므로 극이 아닌 자양이 된다.

### 기신

**① 상관(傷官) — 상관견관(傷官見官)**

> "상관견관, 화기백단(傷官見官 禍起百端) — 상관이 정관을 보면 온갖 화가 시작된다"

상관은 정관을 직접 극하는 오행이다. 정관격에서 상관은 가장 치명적인 기신이다.

**② 겁재(劫財)**: 재성이 용신일 때 재성을 탈취(겁재탈재)하여 용신을 파괴한다.

**③ 관살혼잡(官殺混雜)**: 칠살이 정관과 함께 천간에 나타나 정관의 순수성을 훼손한다.

---

## 4. 성격(成格)

### 유형 ① 재생관(財生官)

재성이 정관을 생하는 구조. 재물로 명예를 이루는 형상이다.

```
구조: 재성(財) →생→ 정관(官) →극→ 일간(我)

예시: 甲木 일간, 酉월(辛金 정관 투출), 천간에 戊土 편재
      戊土가 辛金을 생(土生金) → 재생관 완성
```

### 유형 ② 관인상생(官印相生)

정관이 인성을 생하고 인성이 일간을 생하는 구조. 문관(文官)의 명에서 자주 나타난다.

```
구조: 정관(官) →생→ 인성(印) →생→ 일간(我)

예시: 甲木 일간, 酉월(辛金 정관 투출), 천간에 壬水 정인
      辛金이 壬水를 생(金生水), 壬水가 甲木을 생(水生木)
```

---

## 5. 패격(敗格)

### 유형 ① 상관견관

```
예시: 甲木 일간, 酉월(辛金 정관), 천간에 丁火 상관
      丁火가 辛金을 극(火克金) → 정관 파괴 → 패격
```

자평진전은 "상관이 하나라도 있으면 정관격은 손상된다"고 본다.

### 유형 ② 겁재탈재

```
예시: 甲木 일간, 酉월(辛金 정관), 용신 戊土 재성
      천간에 乙木 겁재 → 乙木이 戊土를 극(木克土)
      → 용신 재성 파괴 → 패격
```

### 유형 ③ 관살혼잡

```
예시: 甲木 일간, 酉월(辛金 정관), 천간에 庚金 칠살
      辛金(정관)과 庚金(칠살) 동시 존재 → 관살혼잡 → 패격
```

---

## 6. 구격(救格) — 패격에서 회생

### 유형 ① 상관패인(傷官佩印)

상관이 정관을 극할 때, 인성이 상관을 제어하여 구격한다.

```
예시: 甲木 일간, 酉월, 丁火 상관(기신), 壬水 정인
      壬水가 丁火를 극(水克火) → 상관 제어 → 구격
      → 패중취귀(敗中取貴)
```

> "상관견관이라도 인성이 있으면 구한다"

### 유형 ② 합거기신(合去忌神)

기신이 다른 오행과 천간합을 이루어 기신 기능을 잃을 때 구격이 된다.

```
예시: 丁火 상관이 壬水와 壬丁합 → 丁火의 상관 기능 약화 → 구격
```

### 유형 ③ 합살유관(合殺留官)

관살혼잡 상황에서 칠살이 합거되어 정관만 남을 때 순수 정관격으로 회생한다.

---

## 7. 격국의 고저(高低)

**높은 정관격 조건**: 정관이 월령에 뿌리를 두고 천간 투출, 용신(재·인)이 힘 있고 뿌리 있음, 기신이 전혀 없거나 완전히 제어됨

**낮은 정관격 조건**: 정관이 투출했으나 지지 뿌리 없어 허약, 용신이 있으나 뿌리 약함, 기신이 완전히 제어되지 않음

---

## 8. 학파 간 이견

**이견 ①** 상관견관의 절대성: 자평진전은 상관이 있으면 반드시 손상이라 하지만, 현대 일부 학파는 상관이 지지에만 있거나 세력이 약하면 정관격이 성립할 수 있다고 본다.

**이견 ②** 관살혼잡 처리: 자평진전은 합거나 제살이 필요하다고 하지만, 현대 실전파 일부는 신강한 사주에서 관살혼잡도 감당 가능하다는 견해를 취한다.

**이견 ③** 정관의 개수: 자평진전은 하나면 족하다고 하지만, 적천수 계열은 신강하면 정관이 둘이어도 가능하다고 본다.

---

## 9. 정관격 핵심 요약

성격: 재생관 또는 관인상생이 이루어지고 기신 없음
패격: 상관견관 또는 겁재탈재 또는 관살혼잡
구격: 상관패인(인성이 상관 제어) 또는 합거기신
