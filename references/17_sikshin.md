# 자평진전(子平眞詮) — 식신격(食神格)

> 자평진전 격국론 시리즈 7편. 식신격은 복록(福祿)과 표현의 격이다.

---

## 1. 정의와 본질

식신격(食神格)은 **월지 장간 중 식신(食神)이 천간에 투출**하여 성립한다.
식신(食神)이란 일간이 음양이 다르게 생(生)하는 오행이다 — 내가 낳는 오행 중 음양이 다른 것.

| 일간 | 식신 |
|---|---|
| 甲(양목) | 丙(양화) → 실은 식신은 음양이 같음 |

잠깐, 정확히 정리하면:
- 식신(食神): 일간이 생(生)하는 오행 중 **음양이 같은** 것
- 상관(傷官): 일간이 생(生)하는 오행 중 **음양이 다른** 것

| 일간 | 식신 | 상관 |
|---|---|---|
| 甲(양목) | 丙(양화) | 丁(음화) |
| 乙(음목) | 丁(음화) | 丙(양화) |
| 丙(양화) | 戊(양토) | 己(음토) |
| 丁(음화) | 己(음토) | 戊(양토) |
| 戊(양토) | 庚(양금) | 辛(음금) |
| 己(음토) | 辛(음금) | 庚(양금) |
| 庚(양금) | 壬(양수) | 癸(음수) |
| 辛(음금) | 癸(음수) | 壬(양수) |
| 壬(양수) | 甲(양목) | 乙(음목) |
| 癸(음수) | 乙(음목) | 甲(양목) |

**식신의 본질**: 표현·생산·복록·의식주를 상징한다. 내 에너지(일간)가 자연스럽게 흘러나오는 표현 오행이다. 상관보다 순하고 안정적이어서 "복신(福神)"이라 불린다.

**식신격의 두 가지 대표 구조**:
1. 식신생재(食神生財) — 표현 능력으로 재물을 만드는 구조
2. 식신제살(食神制殺) — 칠살을 제어하는 구조

---

## 2. 성립 조건

월지 장간 중 식신이 천간에 투출하면 격이 성립한다.

**식신격의 전제**: 식신이 건강하게 유지되어야 한다. 편인(偏印)이 식신을 극하는 효신도식(梟神盜食)이 발생하지 않아야 한다.

---

## 3. 용신·희신·기신

### 용신

**① 재성(財星) — 식신생재(食神生財)**

식신이 재성을 생하는 구조. 내 표현 능력으로 재물을 만드는 가장 자연스러운 흐름이다.

```
구조: 일간 →설기→ 식신(食) →생→ 재성(財)
```

**② 칠살(七殺) — 식신제살(食神制殺)**

식신이 칠살을 극(剋)하는 구조. 칠살격에서 언급한 식신제살과 동일하지만, 이쪽은 식신격의 관점에서 본 것이다. 이 경우 칠살이 오히려 격을 완성시키는 역할을 한다.

```
구조: 식신(食) →극→ 칠살(殺) →극→ 일간(我)
      식신이 살을 막아 일간을 보호
```

### 기신

**① 편인(偏印) — 효신도식(梟神盜食)**

편인이 식신을 극하면 식신의 생산·표현 기능이 완전히 박탈된다. 이것이 식신격에서 가장 치명적인 기신이다.

**② 관성(正官) 과다**

식신격에서 정관이 있으면 식신이 정관을 극하는 상관견관의 문제가 아니지만, 관성이 식신을 제어하거나 격을 혼탁하게 만들 수 있다.

**③ 칠살과 편인이 동시에 존재**

칠살이 있는데 편인까지 있으면, 식신이 이중으로 압박받는다. 편인이 식신을 극하면 칠살을 제어하는 기능도 사라진다.

---

## 4. 성격(成格)

### 유형 ① 식신생재(食神生財) — 복록(福祿)의 명

```
예시: 甲木 일간, 午월(丙火 식신 투출), 천간에 戊土 편재
      甲→丙(식신)→戊(편재) → 식신생재 완성
      내 표현 능력으로 재물을 만드는 형상 → 복록 있는 명
```

자평진전은 "식신생재는 복록이 있다"고 표현한다.

### 유형 ② 식신제살(食神制殺) — 권위(權威)의 명

```
예시: 甲木 일간, 午월(丙火 식신), 천간에 庚金 칠살
      丙火가 庚金을 극(火克金) → 살이 제어됨
      → 식신제살 완성 → 권위 있는 명
```

### 유형 ③ 식신독용(食神獨用) — 순수 표현

재성도 칠살도 없이 식신만 순수하게 작용할 때, 일간의 에너지가 자유롭게 표현되는 복록의 구조.

---

## 5. 패격(敗格)

### 유형 ① 효신도식(梟神盜食)

```
예시: 甲木 일간, 午월(丙火 식신), 천간에 壬水 편인
      壬水가 丙火를 극(水克火) → 식신 기능 박탈 → 패격
      복록·표현·의식주 모두 손상
```

### 유형 ② 식신생재에서 겁재탈재

식신이 생한 재성을 겁재가 탈취하면 식신생재의 결실이 사라진다.

```
예시: 식신이 재성을 생하는데 겁재가 재성을 극
      → 노력해도 재물이 남지 않는 형상 → 패격
```

### 유형 ③ 살강식약(殺强食弱)

식신제살 구조에서 칠살이 식신보다 훨씬 강하면, 식신이 살을 제어하지 못하고 오히려 살에 압도된다.

---

## 6. 구격(救格)

### 유형 ① 재제편인(財制偏印)

효신도식 상황에서 재성이 편인을 극하여 식신을 보호한다.

```
예시: 壬水 편인이 丙火 식신을 극하려 할 때
      戊土 재성이 壬水를 극(土克水) → 편인 제어 → 식신 보호 → 구격
```

### 유형 ② 인성제살보식(印星制殺保食)

살이 너무 강하여 식신이 살을 제어하지 못할 때, 인성이 살을 설기하여 살의 기세를 줄여준다.

---

## 7. 식신격의 특수 구조

**식신패살(食神佩殺)**: 식신격에 칠살이 있을 때 식신이 살을 제어하는 구조. "패(佩)"는 찬다는 의미 — 식신이 살을 차고(제어하고) 다니는 형상.

**식신생재관(食神生財官)**: 식신→재성→관성의 연속 흐름. 능력(식신)으로 재물(재성)을 만들고 재물이 명예(관성)로 이어지는 이상적 흐름.

---

## 8. 식신격과 상관격 비교

| | 식신격 | 상관격 |
|---|---|---|
| 표현 방식 | 순하고 안정적 | 예리하고 반항적 |
| 관성과의 관계 | 식신은 관성을 직접 극하지 않음 | 상관은 정관을 직접 극(상관견관) |
| 복록 | 복신(福神)으로 복록 있음 | 복록보다 능력·재능 강조 |
| 기신 | 편인(효신도식) | 정관(상관견관), 인성 |
| 칠살 관계 | 식신제살 — 안정적 제어 | 상관제살 — 불안정한 제어 |

---

## 9. 학파 간 이견

**이견 ①** 식신제살에서 재성의 역할: 자평진전은 식신제살에서 재성이 없어야 식신의 기능이 온전하다고 하지만, 일부 학파는 재성이 있어도 식신이 살을 제어하면 격이 성립한다고 본다.

**이견 ②** 효신도식의 구제 조건: 재성이 편인을 제어하면 구격이 된다는 점은 공통이지만, 그 세력 비교 기준이 학파마다 다르다.

---

## 10. 식신격 핵심 요약

핵심 본질: 표현·생산·복록의 격
성격: 식신생재(복록) 또는 식신제살(권위)
패격: 효신도식(편인이 식신 극) 또는 겁재탈재
구격: 재성이 편인을 제어하여 식신 보호
