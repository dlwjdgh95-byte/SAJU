# 28. 운 점수화 모델 (Scoring Model)

대운·세운·월운·일운을 정량 점수로 환산하는 rubric. 격국·조후·합충·병약 판단(Step 4~9)을 **먼저 완료한 뒤** 적용한다. 점수는 판단의 요약이지 판단의 대체가 아니다.

---

## 1. 층위 가중 원칙 (Layer Weights)

전체 명운에서 각 층위가 차지하는 비중. 하위 층위는 상위 층위의 기조를 뒤집지 못한다.

| 층위 | 비중 | 역할 |
|---|---|---|
| 원국 | 55 | 판의 근본, 가능성의 총량 |
| 대운 | 25 | 10년 단위 계절 변화 — 용신·기신 좌표 이동. 운 중 압도적으로 무거움 |
| 세운 | 12 | 대운 배경 위에서 그해의 주제·사건 촉발 |
| 월운 | 5 | 강도 약함. 사건 발현 시점(timing) 포착 |
| 일운·시운 | 3 | 미미. 택일·활동 적합도 참고용 |

**층위 합성 규칙 — 체감운(體感運)**: 모든 운 채점 출력에는 층위점수와 함께 **체감운을 반드시 병기**한다. 체감운은 원국 기저점수를 포함한 비중 비례 블렌딩이다.

```
원국 기저점수 = 원국 자체를 5축으로 채점한 값 (명조당 1회 산출, 고정)

체감(대운) = (원국×55 + 대운×25) / 80
체감(세운) = (원국×55 + 대운×25 + 세운×12) / 92
체감(월운) = (원국×55 + 대운×25 + 세운×12 + 월운×5) / 97
```

- 체감운은 진폭이 좁게 나오는 것이 **의도된 설계**다 — 운은 원국이라는 그릇 안의 변동이며, 원국을 뒤집지 못한다.
- 따라서 체감운에는 §3 판정 밴드를 적용하지 않는다. 체감운은 **같은 명조 내 시기 간 상대 비교** 전용이며, 절대 판정은 층위점수로 한다.
- 원국 기저가 낮은 명조는 "평 운세도 체감상 흉"으로 겪을 수 있다 — 층위점수와 체감의 괴리 자체가 해석 정보다.

## 2. 층위 내 5축 채점

각 축에 s ∈ [-1.0, +1.0] 점수 부여 (0.1 단위).

| 축 | 가중치 | 판단 근거 |
|---|---|---|
| ① 격국 (용신·기신 관계) | **W_격 = 45 − W_감** (22~32) | SKILL.md 7-3 표. 용신 도래 +, 기신 강화 −, 구신 합거 − |
| ② 조후 | 18 | 궁통보감 일간별 파일. 조후 용신 보강 +, 조열/한습 악화 − |
| ③ 합충 이벤트 | 22 | 7-4 규칙. 기신 합거·충거 +, 용신 뿌리 충파·삼합 변질 −, 월지 합거 특별 감점 |
| ④ 일간 감당력 | **W_감 = 13~23 (동적)** | 뿌리 보전·재다신약·설기 심화 여부 |
| ⑤ 왕쇠·신살 | 15 (운성 9 + 신살 6) | 아래 §2-1 |

### 감당력 동적 가중 — 신강도 지수 S

신약할수록 감당력 축이 무거워지고, 그만큼 격국 축이 가벼워진다.

```
S = 일간 통근점수(록왕 3 / 생지 2 / 여기 1 / 묘고 0.5, 합충 손상 시 감점)
  + 월령 득기 (+2)
  + 천간 인성·비겁 생조 (각 +1)
  (상한 10)

W_감 = clamp(23 − S, 13, 23)
W_격 = 45 − W_감
```

- 극신약(S=0) → W_감 23 / W_격 22
- 신강(S≥10) → W_감 13 / W_격 32

### 2-1. ⑤축 왕쇠·신살 세부

**12운성 (배점 9)**: 일간 기준, 해당 층위 지지의 왕쇠 단계.
- 장생·건록·제왕 등 왕지 계열 +, 병·사·절 등 쇠지 계열 −, 묘·태·양은 질감 위주 소폭
- 점수보다 **질감 해석**이 본령: 묘=저장·침잠, 절=단절·재출발, 장생=개화 등. 순세 태그 판정에 우선 반영.

**12신살·공망 (배점 6)**: 공망(일주 기준 — 견고), 화개·역마·도화·육해 등 핵심 신살 위주.
- 12신살은 년지 기준을 기본으로 하되 일지 기준 이견이 있음을 인지 (23번 파일 층위)
- 신살은 길흉 방향보다 **사건의 색채**(고독·이동·주목·정체)로 읽고, 점수 기여는 보수적으로.

**⚠️ 이중 계상 방지 (필수)**: 운성·신살 판정이 다른 축에서 이미 채점된 사실의 재진술이면 해당 s를 **절반으로 감쇠**한다.
- 예) 癸 일간에게 未=묘·午=절은 "화토 지지에 물 뿌리 없음"의 재진술 → ④축과 중복 → 감쇠 적용
- 예) 화개가 삼합 고지(庫地)에서 발동하는 경우 ③축 삼합 판정과 중복 → 감쇠 적용
- 신살이 다른 축에 없는 독립 정보(공망, 도화, 역마 등)일 때만 온전히 계상.

### 2-2. 합충축 판정 전 필수 선행 체크

**① 충산합해(沖散合解) 우선 판정**: 세운·월운의 충을 채점하기 전에, 충당하는 글자가 **상위 층위(원국·대운)의 합(삼합·육합)에 묶여 있는지** 먼저 확인한다.
- 묶여 있으면 → 그 충은 손상이 아니라 **합의 해소** — 묶였던 글자(특히 일간 뿌리)가 복구된다. ③축 부호를 반전(+)하고 ④감당력에도 회복 가산.
- 자유로운 글자에 대한 충 → 기존 7-4 규칙대로.
- ⚠️ 같은 글자를 상위 층위에서 "합 변질 감점", 하위 층위에서 "충파 감점"으로 이중 처벌하지 않는다.

**② 생 사슬(통로) 보정**: 층위 기둥이 천간→지지 상생(식신생재, 재생관 등)으로 원국 주 세력의 **출구**를 여는 구조면, 십신 개별 길흉 합산보다 흐름을 우선한다.
- 예) 식상 과다가 병인 명조에 식신생재 기둥 세운 → 설기가 순환·결실로 전환. ①격국축을 통로 관점으로 재채점(기신 개별 감점 대신 순세 가산).
- 단, 일간 뿌리가 살아 있을 때만 — 뿌리 없는 상태의 생재는 재다신약 가속이므로 보정 금지.

### 2-3. 불발 합의 세운 점화 (세운 층위까지만 적용)

원국·대운에서 합이불화(合而不化)로 판정된 천간합은 **화신(化神)이 왕한 세운이 들어오는 해에 한해 부분 점화**될 수 있다.
- 예) 원국 丙 × 대운 辛의 丙辛합이 불화 상태일 때, 세운 지지가 亥·子(수왕지)면 그 해에 한해 합화수가 부분 성립 → ③합충축에 화신 오행의 길흉으로 가산
- "부분"의 의미: 원국 구조 자체가 바뀌는 진화(眞化)가 아니라 그 해의 기운 흐름으로만 계상. 세운이 지나면 원상 복귀.
- **월운 이하에는 적용하지 않는다** — 월운의 강도로는 합화를 점화하지 못한다.

### 점수식

```
천간점수 = 50 + Σ(sᵢ(천간 평가) × Wᵢ) / 2
지지점수 = 50 + Σ(sᵢ(지지 평가) × Wᵢ) / 2

전반 = 천간점수 × 0.7 + 지지점수 × 0.3
후반 = 천간점수 × 0.3 + 지지점수 × 0.7
종합 = (전반 + 후반) / 2               (모두 0~100 클램프)
```

- 천간·지지는 시기별 **주도권**의 차이지 단독 작용이 아니다. 지지는 10년 내내 깔려 있고 천간은 10년 내내 떠 있다 — 고립 채점 금지.
- **절각(截脚) 감쇠**: 층위 천간이 원국·해당 층위 지지 어디에도 통근하지 못하고, 좌지(坐支)의 생조도 받지 못하면 천간 s에 ×0.7. (통근 없는 천간은 뿌리 없는 나무)
- **개두(蓋頭) 완화**: 층위 천간이 좌지의 기신 작용을 극으로 눌러줄 때 지지 s를 ×0.85로 완화 가능 (보수적 적용).

## 4. 순세 태그 (필수 병기)

점수는 길흉 총량일 뿐, "이 운으로 무엇을 하는가"는 별도다. 매 운마다 태그 1개를 병기한다.

| 태그 | 조건 | 활용 방향 |
|---|---|---|
| 발산기 | 식상·비겁 강화 | 창작·표현·시작. 결실 기대는 금물 |
| 축적기 | 인성·습토·용신 도래 | 공부·자격·기반 구축. 최대 투자 구간 |
| 수렴기 | 재성·관성 안정 작동 | 계약·결실·확정. 벌린 것을 거둠 |
| 전환기 | 충·형·월지 동요 | 이동·이직·구조 변경. 안정 기대 금물 |
| 방어기 | 기신 강화 + 뿌리 손상 | 현상 유지·건강·손절 판단. 확장 금지 |

## 5. 일운 섹터 점수

일운은 종합 점수 대신 **활동별 적합도(1~5)**로 출력한다. 기본값 3에서 일진 천간·지지의 특징으로 가감 후 1~5 클램프. 상위 층위(세운·월운) 기조가 흉이면 4~5점도 "그 기조 안의 좋은 날"로 해석한다.

**7개 섹터**: 계약·문서 / 모임·관계 / 여행·이동 / 이사·개업 / 창작·표현 / 재물·투자 / 휴식·회복

### 매핑 원칙 (십신·합충·신살 → 섹터)

| 신호 | 가감 |
|---|---|
| 인성일 (용신이면 가중) | 계약·문서 +2, 재물 판단 +2, 휴식 +1 |
| 식상일 | 창작·표현 +2 (신약이면 휴식 −) |
| 비겁일 | 모임 +1~2, 뿌리 보강 시 휴식 + |
| 재성일 (기신이면) | 재물 −2 (지출 충동), 모임(이성) +1 |
| 일간 합·충 일진 | 계약·문서 −2 (판단 묶임/동요) |
| 공망일 | 계약·이사 −2, 휴식(비움) +1 |
| 도화 | 모임·관계 +2 |
| 역마·지살 | 여행·이동 +2 |
| 천을귀인 | 계약·문서 +1 |
| 문창·화개 | 창작·표현 +1~2 |
| 충·형·파 발생일 | 이사·개업 −2, 계약 −1 (단 충산합해·기신 충거는 예외) |
| 삼형 완성일 | 이사·개업·계약 강한 감점 |

- 판정 전 §2-2 선행 체크(충산합해·통로)를 일 단위에도 적용한다.
- 합화 점화(§2-3)는 일운에 적용하지 않는다.
- 출력은 달력 형식 권장: 일별 간지 + 최고 섹터 + 경고 섹터.

## 6. 주의

- 점수가 같아도 태그가 다르면 전혀 다른 운이다. 점수 단독 인용 금지.
- 삼합·충으로 일간 유일 뿌리가 손상되는 경우 ③·④축에 **중복 반영**한다 (뿌리는 이벤트이자 감당력이므로 이중 계상이 의도된 설계다).
- rubric이 실제 임상 감각과 어긋나면 축 점수(s)가 아니라 **가중치·조건을 교정**하고 SKILL.md errata에 기록한다.

### 7-1. 확인된 분업과 미해결 과제

- **분업 확인**: 사건의 종류(갇힘·연애·시험·이동)는 ⑤신살축이, 크기는 ①격국·④감당력축이 담당. 2017 재살=영창, 2018 도화=연애가 대표 히트.
- **미해결 ①**: 초년 乙酉·甲申은 층위점수 51~52(평)인데 체감은 흉. 원국 기저(43) 병기로 부분 설명되나 잔여 격차 존재. 한 사례에 맞춘 규칙 추가(과적합) 금지 — 타 명조 백테스트 누적 후 재검토.
- **미해결 ②**: 2025 잔여 격차(54 vs 체감 대길). "통로 완성+뿌리 복구 동시 발생 가산" 조항은 보류 상태.
