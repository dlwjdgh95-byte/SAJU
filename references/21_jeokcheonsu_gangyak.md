# 적천수(滴天髓) — 강약론·종격(從格) 판단 규칙

> 이 문서는 적천수의 핵심 이론인 일간 강약 판단법과 종격(從格) 성립 조건을 정리한 것이다.
> 자평진전 격국론과 병용할 때, 격국 판단 이전에 "이 사주가 일반격인가 종격인가"를 먼저 가려내는 데 사용한다.

---

## 1. 적천수 강약론의 기본 원리

### 1-1. 적천수의 핵심 명제

적천수는 일간(日干)의 강약을 판단하는 데 가장 중요한 기준으로 **월령 득기 여부**와 **지지 통근의 질(質)**을 제시한다.

```
"天干之勢 在於動 地支之勢 在於靜"
— 천간의 세력은 유동적이고, 지지의 세력은 고정적이다.

"旺衰 非看月令 而不是 月令爲定"
— 강약은 월령을 보되, 월령만으로 결정하지 않는다.
```

### 1-2. 일간 강약 판단의 4대 요소

| 순서 | 요소 | 내용 | 비중 |
|---|---|---|---|
| ① | 월령 득기(得氣) | 일간이 월지에서 생조·비화를 받는가 | 최대 (약 40%) |
| ② | 통근(通根) | 일간과 같은 오행이 지지에 몇 개, 어떤 질로 있는가 | 핵심 (약 30%) |
| ③ | 천간 생조(生助) | 인성·비겁이 천간에서 일간을 돕는가 | 보조 (약 20%) |
| ④ | 전체 세력 균형 | 극설(克洩)하는 오행 대비 생조하는 오행의 비율 | 미세조정 (약 10%) |

### 1-3. 월령 득기(得氣) 판단 기준

```
일간이 월지에서 得氣하는 경우:
- 월지가 일간과 같은 오행 (비화): 예) 甲일간 + 寅월
- 월지가 일간을 생하는 오행 (인성): 예) 甲일간 + 亥월 (水生木)

일간이 월지에서 失氣하는 경우:
- 월지가 일간을 극하는 오행: 예) 甲일간 + 申월 (金克木)
- 월지가 일간이 극하는 오행: 예) 甲일간 + 辰월 (木克土, 일간 소모)
- 월지가 일간이 생하는 오행: 예) 甲일간 + 巳월 (木生火, 설기)
```

### 1-4. 통근(通根)의 질적 평가

적천수는 통근을 단순히 개수로 세지 않고, **질(質)**을 따진다.

```
강한 통근 (진근, 眞根):
- 일간과 동일한 오행의 본기(本氣): 예) 甲일간 + 寅(본기 甲)
- 월지에 통근: 월령의 힘을 직접 빌림

보통 통근:
- 일간과 동일한 오행의 중기(中氣): 예) 甲일간 + 亥(중기 甲)
- 일간과 동일한 오행의 여기(餘氣): 예) 甲일간 + 辰(여기 乙 → 甲의 동류)

약한 통근 (허근, 虛根):
- 조토(燥土: 未·戌)에 묻힌 장간: 계절 기운에 의해 실력 약화
- 충(沖)당하는 지지의 장간: 뿌리가 흔들려 제 기능 못함
- 합거(合去)되어 변질된 지지의 장간: 본래 오행 기능 상실
```

### 1-5. 통근 개수별 강약 대략 기준

```
통근 0개: 무근(無根) — 극약, 종격 후보
통근 1개 (약한 통근): 약(弱) — 종격 가능성 검토
통근 1개 (강한 통근): 약~중 — 일반격 가능
통근 2개: 중(中) — 일반격
통근 3개 이상: 강(强) — 신강 사주

※ 이것은 대략적 기준이며, 월령 득기 여부와 천간 생조를 종합해야 한다.
```

## 2. 신강(身强)·신약(身弱) 판단 프로세스

```
Step 1: 월령 득기 여부 확인
        → 得氣: 신강 쪽으로 기본 가산
        → 失氣: 신약 쪽으로 기본 가산

Step 2: 지지 통근 수와 질(質) 확인
        → 본기 통근 2개 이상 + 월령 득기 → 확정적 신강
        → 통근 0개 + 월령 실기 → 극약 (종격 검토)

Step 3: 천간 생조 확인
        → 인성·비겁이 천간에 2개 이상: 신강 방향
        → 관살·식상·재성이 천간에 2개 이상: 신약 방향

Step 4: 전체 세력 비교
        → 일간을 생조·비화하는 오행(인성+비겁) vs 극설하는 오행(관살+식상+재성)
        → 비율이 3:5 이하면 신약, 5:3 이상이면 신강
        → 4:4 전후면 중화(中和) — 가장 좋은 균형

Step 5: 최종 판단
        → 신강: 식상·재성·관살이 용신 후보
        → 신약: 인성·비겁이 용신 후보
        → 극강/극약: 종격 여부 검토 (아래 3장)
```

## 3. 종격(從格) — 성립 조건과 분류

### 3-1. 종격의 기본 원리

```
"旺之極者 不可逆也 從其旺神 可也"
— 극도로 왕성한 것은 거스를 수 없으니, 그 왕신을 따르는 것이 옳다.

적천수의 핵심 통찰:
일간이 너무 약하여 자신의 오행을 유지할 수 없을 때,
강한 오행을 따라가는(從) 것이 오히려 길한 명이 된다.
단, 종격이 성립하려면 엄격한 조건을 충족해야 한다.
```

### 3-2. 종격 성립의 절대 조건

```
필수 조건 (하나라도 미충족 시 종격 불성립):

① 일간 무근(無根) 또는 극약근(極弱根)
   - 일간과 같은 오행이 지지 4자에 본기로 하나도 없음
   - 또는 있더라도 충(沖)·합거(合去)로 뿌리 기능 상실
   
② 월령 실기(失氣)
   - 월지가 일간을 극하거나 설기하는 오행
   
③ 천간에 인성·비겁이 없거나 무력
   - 인성이 있어도 지지에 통근하지 못하면 무력
   - 비겁이 있어도 합거되어 기능을 잃으면 무력

④ 특정 오행이 사주를 압도적으로 지배
   - 지지 4자 중 3자 이상이 동일 계열
   - 또는 천간·지지 합산 8자 중 5자 이상이 동일 계열
```

### 3-3. 종격의 분류

#### (1) 종재격(從財格)

```
조건:
- 일간 무근 또는 극약근
- 재성(財星)이 사주를 지배: 지지에 재성 오행 3자 이상
- 인성 없거나 무력 (인성이 있으면 일간을 도와 종하지 못함)
- 비겁 없거나 무력

용신: 재성, 식상(재를 생함)
기신: 인성(종을 깨뜨림), 비겁(종을 깨뜨림), 관살(재를 설기함 — 학파 이견 있음)

특징:
- 재물·사업·현실적 성취에 강한 명
- 대운에서 인성·비겁이 오면 종이 깨져 대흉
```

#### (2) 종살격(從殺格) / 종관격(從官格)

```
조건:
- 일간 무근 또는 극약근
- 관살(官殺)이 사주를 지배
- 식상 없거나 무력 (식신이 관살을 제어하면 종하지 못함)
- 인성이 있으면 오히려 화살(化殺) 역할 → 종살격 보조

용신: 관살, 재성(관살을 생함)
기신: 식상(관살을 극함 → 종을 깨뜨림), 비겁

특징:
- 종살격에서 인성이 있으면 "화살(化殺)"로 관살의 극한 압박을 부드럽게 전환
  → 이때 인성은 기신이 아니라 희신
- 식상이 오면 관살을 건드려 종이 깨지므로 대흉
```

#### (3) 종아격(從兒格) / 종식상격(從食傷格)

```
조건:
- 일간 무근 또는 극약근 (또는 일간이 중간 강도라도 식상이 압도적)
- 식상(食傷)이 사주를 지배
- 인성 없거나 무력 (인성이 식상을 극하면 종하지 못함)

용신: 식상, 재성(식상의 결실)
기신: 인성(식상을 극함), 관살(식상과 충돌)

특징:
- 종아격은 식신·상관의 기운을 따르는 것
  → 표현력·예술·기술·학문에 강한 명
- "아(兒)"는 식상을 뜻함 (내가 낳은 것)
```

#### (4) 종강격(從强格) / 종왕격(從旺格)

```
조건:
- 일간이 극강(極强): 비겁·인성이 사주를 지배
- 재성·관살·식상이 없거나 극히 약함
- 일간이 너무 강해서 극설(克洩)할 수 없는 상태

용신: 비겁, 인성 (강한 기운을 따름)
기신: 관살(일간을 극하려 하나 역극당함), 재성(약한 재성이 강한 비겁에 탈취됨)

종강격과 종왕격의 차이 (학파에 따라 구분):
- 종강격: 비겁 위주로 강한 경우
- 종왕격: 인성 위주로 강한 경우
- 실전에서는 합쳐서 "종강격"으로 부르는 경우가 많음
```

#### (5) 화기격(化氣格) — 특수 종격

```
조건 (매우 엄격):
- 일간이 인접 천간과 천간합(天干合)을 이룸
- 합화(合化)한 결과 오행이 월령의 지지를 받음
- 합화 오행 외의 다른 오행이 사주에 거의 없음

5종 화기격:
| 합 | 화(化) 오행 | 격명 | 월령 조건 |
|---|---|---|---|
| 甲己합 | 土 | 화토격 | 辰戌丑未월 |
| 乙庚합 | 金 | 화금격 | 申酉월 |
| 丙辛합 | 水 | 화수격 | 亥子월 |
| 丁壬합 | 木 | 화목격 | 寅卯월 |
| 戊癸합 | 火 | 화화격 | 巳午월 |

용신: 합화된 오행, 합화 오행을 생하는 오행
기신: 합화를 깨뜨리는 오행 (합화 오행을 극하거나, 합을 쟁합하는 글자)

⚠️ 화기격은 성립 조건이 극히 까다로워 실전에서 드물다.
   조건을 하나라도 위반하면 가화(假化) — 진짜 종격이 아님.
```

## 4. 종격 성립 판단 프로세스 (실전)

```
Step 1: 일간 강약 초기 판단
        → 일간이 극약한가? (통근 0~1, 월령 실기, 천간 생조 없음)
        → YES → Step 2로
        → NO → 일반격 (격국론 적용)

Step 2: 사주 지배 오행 확인
        → 지지 4자 중 어느 오행이 3자 이상을 차지하는가?
        → 천간까지 합산 시 어느 오행이 5자 이상인가?
        → 지배 오행 확인 → Step 3으로

Step 3: 종격 파괴 요소(破格 要素) 점검
        → 일간을 생조하는 오행(인성·비겁)이 천간·지지에 있는가?
        → 있다면: 그 오행이 통근하고 있는가? 합거되지 않았는가?
        → 통근하고 힘 있으면 → 종격 불성립 → 일반격으로 복귀
        → 통근 없이 떠 있기만 하면 → 종격 성립 가능 (무력한 글자)

Step 4: 종격 유형 분류
        → 지배 오행이 재성 → 종재격
        → 지배 오행이 관살 → 종살격
        → 지배 오행이 식상 → 종아격
        → 지배 오행이 비겁·인성 → 종강격

Step 5: 진종(眞從) vs 가종(假從) 판단
        → 파괴 요소가 완전히 없음: 진종(眞從) — 종격 확정, 대운 길흉 판단 명확
        → 파괴 요소가 미약하게 있음: 가종(假從) — 종격으로 보되 불안정
          → 가종은 대운에서 인성·비겁이 오면 종이 쉽게 깨져 급변

Step 6: 용신·기신 결정
        → 종격 유형에 따라 위 3-3의 각 격별 용신·기신 적용
```

## 5. 강약론과 격국론의 관계 — 적용 우선순위

```
판단 순서:
① 먼저 종격 여부를 판단 (적천수 강약론)
   → 종격이면: 종격의 용신·기신 체계 적용 (격국론 미적용)
   → 종격이 아니면: ②로

② 일반격 → 자평진전 격국론 적용
   → 월령 기반 격국 판단 → 용신·기신 결정

③ 격국론 적용 후 병약론 검토
   → 오행 편중이 극단적이면 병약론 우선

④ 조후론으로 보정
   → 격국 용신과 조후 용신의 일치/불일치 확인
```

## 6. 대운에서의 종격 판단

### 6-1. 종격 사주의 대운 길흉

```
종격에서 길운:
- 종하는 오행이 강화되는 대운 → 대길
- 종하는 오행을 생하는 대운 → 길

종격에서 흉운:
- 종을 깨뜨리는 오행(인성·비겁)이 오는 대운 → 대흉
  → 특히 해당 오행이 지지에 통근하면 종이 완전히 깨져 급변
- 가종(假從) 사주에서 미약했던 파괴 요소가 대운에서 힘을 얻으면 → 대흉

⚠️ 종격이 깨지는 대운의 특징:
- 극적인 환경 변화 (직장·거주지·관계의 급변)
- 기존 성공 구조의 붕괴
- 진종보다 가종이 깨질 때 피해가 더 큼 (본래 불안정했으므로)
```

### 6-2. 종격 vs 극약(極弱) 일반격 — 경계선의 판단

```
가장 어려운 판단 지점:
일간이 극히 약하지만 미약한 통근이 1개 있는 경우

종격으로 보는 견해 (적천수파):
- 통근이 1개라도 충·합거·조토 위의 약한 근이면 종으로 본다
- "뿌리가 있어도 기능하지 못하면 없는 것과 같다"

일반격으로 보는 견해 (자평진전파):
- 통근이 1개라도 있으면 일간이 완전히 무너지지 않는다
- 극약 일반격으로 보고 인성·비겁을 용신으로 잡는다

실전 처리:
- 양쪽 가능성을 모두 제시하고, 대운 흐름으로 검증
  → 인성·비겁 대운에서 좋았으면 일반격이 맞음
  → 인성·비겁 대운에서 오히려 나빴으면 종격이 맞음
```

## 7. 주의사항

```
① 종격은 극소수의 사주에만 해당한다.
   일반적으로 사주의 90% 이상은 일반격이다.
   종격을 남발하면 해석이 완전히 틀어진다.

② "신약하다"와 "종격이다"는 전혀 다른 판단이다.
   신약해도 통근이 있고 인성이 있으면 일반격이다.
   종격은 일간이 자기 오행을 유지할 수조차 없는 극단적 상태다.

③ 화기격은 실전에서 거의 나오지 않는다.
   조건이 하나라도 틀리면 가화(假化)이며, 가화는 종격보다 불안정하다.
   화기격으로 단정하기 전에 반드시 일반격 가능성을 먼저 검토한다.

④ 종격과 억부론의 관계:
   종격에서는 억부론이 역전된다.
   - 일반격: 신약하면 인성·비겁으로 보강 (억부론)
   - 종격: 신약해도 인성·비겁이 기신 (종하는 오행을 따름)
   → 이 역전을 모르면 용신과 기신을 정반대로 잡는 치명적 오류
```
