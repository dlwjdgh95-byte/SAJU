# 자평진전(子平眞詮) — 상관격(傷官格)

> 자평진전 격국론 시리즈 8편. 상관격은 팔격 중 가장 복잡하고 변수가 많은 격이다.

---

## 1. 정의와 본질

상관격(傷官格)은 **월지 장간 중 상관(傷官)이 천간에 투출**하여 성립한다.
상관(傷官)이란 일간이 음양이 다르게 생(生)하는 오행이다.

| 일간 | 상관 | 식신 |
|---|---|---|
| 甲(양목) | 丁(음화) | 丙(양화) |
| 乙(음목) | 丙(양화) | 丁(음화) |
| 丙(양화) | 己(음토) | 戊(양토) |
| 丁(음화) | 戊(양토) | 己(음토) |
| 戊(양토) | 辛(음금) | 庚(양금) |
| 己(음토) | 庚(양금) | 辛(음금) |
| 庚(양금) | 癸(음수) | 壬(양수) |
| 辛(음금) | 壬(양수) | 癸(음수) |
| 壬(양수) | 乙(음목) | 甲(양목) |
| 癸(음수) | 甲(양목) | 乙(음목) |

**상관의 본질**: 창의성·반항·예술적 재능·날카로운 지성을 상징한다. 정관을 직접 극하는 오행으로 규범과 권위에 도전하는 성질이 있다. 식신보다 예리하고 강렬하다.

**상관격의 핵심 딜레마**: 상관이 정관을 극하므로(상관견관), 정관격의 적이다. 그러나 상관 자체는 강렬한 재능과 에너지의 원천이기도 하다. 이 딜레마를 어떻게 다루느냐가 상관격의 핵심이다.

---

## 2. 성립 조건

월지 장간 중 상관이 천간에 투출하면 격이 성립한다.

**상관격의 복잡성**: 자평진전은 상관격을 하나의 고정 틀로 보지 않는다. 사주 구조에 따라 세 가지 다른 방향으로 용신이 달라진다.

---

## 3. 용신·희신·기신 — 세 가지 유형

상관격은 사주의 다른 구성 요소에 따라 용신이 완전히 달라진다.

### 유형 A: 상관생재(傷官生財) — 재성이 용신

관성이 없고 상관이 재성을 생하는 구조. 날카로운 능력(상관)으로 재물(재성)을 만드는 형상이다.

```
구조: 일간 →설기→ 상관(傷) →생→ 재성(財)
용신: 재성
기신: 겁재(재성 탈취), 관살(상관이 있는데 관살까지 있으면 혼잡)
```

### 유형 B: 상관패인(傷官佩印) — 인성이 용신

관성이 있는 상황에서 상관이 정관을 극하려 할 때, 인성이 상관을 제어하여 정관을 보호하는 구조.

> "상관견관이라도 인성이 있으면 구한다"

```
구조: 인성(印) →극→ 상관(傷) → 상관의 정관 극 제어
      + 관성(官) →생→ 인성(印) →생→ 일간(我)
용신: 인성
기신: 재성(재파인 — 인성을 극파)
```

### 유형 C: 상관상관(傷官傷官) — 상관 자체를 쓰는 구조

관성이 없고 재성도 없으며 상관의 에너지가 순수하게 표출되는 구조. 예술·창작·특수 능력의 형상이다.

```
구조: 일간 →설기→ 상관(傷) → 순수 표현·창의
특징: 관성이 있으면 반드시 제거하거나 상관을 제어해야 함
```

---

## 4. 상관격의 세부 분류 — 오행별 특성

자평진전은 상관격을 일간의 오행에 따라 다섯 가지로 분류한다.

**① 금수상관(金水傷官)**: 경금·신금 일간이 수(水)로 상관

> "금수상관은 관성을 좋아한다 (金水傷官喜見官)"

금수 상관은 다른 상관과 달리 관성이 있어도 문제가 적다. 오히려 관성이 격을 완성하는 경우가 많다.

**② 화토상관(火土傷官)**: 병화·정화 일간이 토(土)로 상관

> "화토상관은 인성을 좋아한다"

화토 상관은 조열(燥熱)의 기운이 강하므로 인성(수·목)으로 조절하는 것이 좋다.

**③ 목화상관(木火傷官)**: 갑목·을목 일간이 화(火)로 상관

> "목화상관은 관살을 꺼린다"

목화 상관은 상관의 기운이 강렬하여 관살이 있으면 충돌이 심하다.

**④ 수목상관(水木傷官)**: 임수·계수 일간이 목(木)으로 상관

재성이 용신이 되는 경우가 많다.

**⑤ 토금상관(土金傷官)**: 무토·기토 일간이 금(金)으로 상관

인성이 용신이 되는 경우가 많다.

## 5. 성격(成格)

### 성격 A: 상관생재 성격

```
예시: 庚金 일간, 子월(癸水 상관 투출), 천간에 甲木 편재
      庚→癸(상관)→甲(편재) → 상관생재 완성
      날카로운 능력(상관)으로 재물을 만드는 형상
```

### 성격 B: 상관패인 성격

```
예시: 庚金 일간, 子월(癸水 상관), 천간에 丁火 정관·戊土 편인
      戊土 편인이 癸水 상관을 극(土克水) → 상관의 정관 극 방지
      丁火 정관이 戊土를 생(火生土) → 관인상생
      → 상관패인 구조 완성
```

### 성격 C: 금수상관 성격

```
예시: 庚金 일간, 子월(癸水 상관), 천간에 己土 정관
      금수상관에서 관성이 오히려 격을 완성 → 성격
```

---

## 6. 패격(敗格)

### 패격 A: 상관생재에서 겁재탈재

상관이 재성을 생하는데 겁재가 재성을 탈취하면 패격이 된다.

### 패격 B: 상관패인에서 재파인(財破印)

인성이 용신인데 재성이 인성을 극(財克印)하면 인성이 파괴되어 패격이 된다.

```
예시: 상관패인 구조에서 천간에 丙火 편재(기신)
      丙火가 戊土(인성)를 생 → 토의 에너지가 흐트러짐
      또는 재성이 직접 인성을 극파 → 패격
```

### 패격 C: 상관견관(傷官見官) — 상관격의 가장 위험한 패격

상관격에 정관이 있을 때, 상관이 정관을 극하여 서로 충돌하는 구조.

```
예시: 甲木 일간, 午월(丁火 상관), 천간에 辛金 정관
      丁火 상관이 辛金 정관을 극(火克金) → 상관견관 → 패격
      (단, 금수상관은 예외)
```

---

## 7. 구격(救格)

### 구격 A: 인성제상(印星制傷) — 상관패인

인성이 상관을 제어하여 상관견관의 충돌을 막는다.

### 구격 B: 합거상관(合去傷官)

상관이 다른 오행과 합거되어 상관의 기능이 약화되면 구격이 된다.

### 구격 C: 거관유상(去官留傷)

상관격에 정관이 있을 때 정관을 제거(합거·극파)하여 순수한 상관격으로 만들면 패격이 해소된다. 다만 이 경우 정관이 완전히 제거되어야 한다 — 정관이 남아있으면 충돌이 지속된다.

---

## 8. 상관격의 특수 구조

**상관제살(傷官制殺)**: 상관이 칠살을 제어하는 구조. 식신제살보다 불안정하지만 유효하다. 칠살이 있는 상관격에서 상관이 살을 제어하면 일종의 성격이 된다.

**순수상관(純粹傷官)**: 상관의 기운만 순수하게 표출되는 구조. 예술·창작·특수 능력으로 빛나는 명.

---

## 9. 학파 간 이견

**이견 ①** 금수상관 특례: 자평진전이 "금수상관희견관"이라 하여 금수 상관의 특례를 인정하지만, 이 해석의 범위에 대해 학파마다 다르다.

**이견 ②** 상관제살의 유효성: 자평진전은 식신제살을 더 안정적으로 보지만, 일부 학파는 상관제살도 동등하게 유효하다고 본다.

**이견 ③** 오행별 상관 분류: 자평진전의 오행별 상관 분류가 지나치게 도식적이라는 비판이 있다. 실전에서는 각 사주의 구체적 구조가 더 중요하다는 견해가 있다.

---

## 10. 상관격 핵심 요약

핵심 복잡성: 용신이 사주 구조에 따라 재성·인성·상관 자체로 달라짐
성격 A: 상관생재 — 날카로운 능력으로 재물 생산
성격 B: 상관패인 — 인성이 상관 제어, 정관 보호
성격 C: 순수상관 — 예술·창의적 표현
패격: 상관견관(상관이 정관 극), 재파인(재성이 인성 극)
구격: 인성이 상관 제어, 또는 정관을 합거하여 충돌 해소
