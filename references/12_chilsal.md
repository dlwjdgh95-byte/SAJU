# 자평진전(子平眞詮) — 칠살격(七殺格)

> 자평진전 격국론 시리즈 2편. 칠살격은 정관격과 대비되는 살(殺)의 논리다.
> "살이 있으면 반드시 제어해야 한다"는 원칙이 이 격 전체를 관통한다.

---

## 1. 정의와 본질

칠살격(七殺格)은 **월지 장간 중 편관(偏官)이 천간에 투출**하여 격이 성립한다.
편관(偏官)이란 일간을 음양이 같게 극(剋)하는 오행으로, 칠살(七殺)이라고도 부른다.

| 일간 | 칠살(편관) |
|---|---|
| 甲(양목) | 庚(양금) |
| 乙(음목) | 辛(음금) |
| 丙(양화) | 壬(양수) |
| 丁(음화) | 癸(음수) |
| 戊(양토) | 甲(양목) |
| 己(음토) | 乙(음목) |
| 庚(양금) | 丙(양화) |
| 辛(음금) | 丁(음화) |
| 壬(양수) | 戊(양토) |
| 癸(음수) | 己(음토) |

**칠살의 본질**: 정관이 법도 안에서 일간을 다스린다면, 칠살은 압박과 강제력으로 일간을 극한다. 제어(制)되지 않으면 흉하고, 제어되면 강력한 권력과 리더십으로 전환된다.

> 고전의 표현: "살이 있으면 반드시 제어해야 귀하고, 제어 없으면 흉하다 (殺有制則貴, 無制則凶)"

---

## 2. 성립 조건

월지 장간 중 편관이 천간에 투출하면 칠살격이 성립한다.

**핵심 원칙 — 칠살의 순수성**

> "살은 하나면 위엄이 있고, 둘이면 폭군이 된다"

칠살도 정관처럼 하나가 이상적이다. 칠살이 여럿이거나 정관이 혼합되면 혼잡해진다.

---

## 3. 용신·희신·기신

### 용신

**① 식신(食神) — 식신제살(食神制殺)**

식신이 칠살을 극(剋)하여 살의 폭력적 기운을 제어한다. 이것이 칠살격에서 가장 이상적인 용신 구조다.

```
구조: 식신(食) →극→ 칠살(殺) →극→ 일간(我)
      식신이 살을 막아 일간을 보호
```

**② 인성(印星) — 인성화살(印星化殺)**

인성이 칠살을 설기(洩氣)하여 살의 기운을 부드럽게 한다. 살이 인성을 생하고(殺生印), 인성이 일간을 생하는(印生我) 구조로 살의 에너지를 유용하게 전환한다.

```
구조: 칠살(殺) →생→ 인성(印) →생→ 일간(我)
      살의 에너지가 인성을 거쳐 일간을 돕는 형상
```

### 기신

**① 재성(財星) — 재성생살(財星生殺)**

> "살격에서 재성은 기신이다. 재가 살을 생하면 살이 더욱 강해진다"

재성이 칠살을 생(財生殺)하면 이미 강한 살이 더욱 맹렬해진다. 칠살격에서 재성은 용신이 될 수 없다.

**② 상관(傷官)**

식신과 달리 상관도 살을 제어하지만, 상관제살은 불안정하다. 상관은 너무 예리하여 살을 지나치게 제어하거나, 반대로 정관이 있을 때 정관을 손상시키는 이중 문제가 있다.

**③ 인성과 식신의 충돌 — 인수화식(印綬化食)**

인성이 용신인 상황에서 식신이 있으면: 인성이 식신을 극(印克食)하여 식신의 제살 기능을 방해. 이 경우 인성과 식신이 서로를 방해하는 혼란이 발생한다.

---

## 4. 성격(成格)

### 유형 ① 식신제살(食神制殺)

```
구조: 식신이 칠살을 제어

예시: 甲木 일간, 申월(庚金 칠살 투출), 천간에 丙火 식신
      丙火가 庚金을 극(火克金) → 살이 제어됨 → 성격
```

이것이 칠살격의 가장 이상적 형태다. "칠살이 있으면 식신이 있어야 귀하다"는 원칙이 여기서 나온다.

### 유형 ② 인성화살(印星化殺)

```
구조: 살이 인성을 생하고 인성이 일간을 생

예시: 甲木 일간, 申월(庚金 칠살), 천간에 壬水 편인
      庚金이 壬水를 생(金生水) → 壬水가 甲木을 생(水生木)
      → 살의 에너지가 인성을 통해 일간을 도움
```

### 유형 ③ 칠살독살(七殺獨殺)

제어 없이도 일간이 칠살을 감당할 만큼 신강(身强)할 때, 살이 오히려 일간의 힘을 더욱 돋보이게 한다. 단, 자평진전은 이 경우를 인정하면서도 신중하게 판단할 것을 요구한다.

---

## 5. 패격(敗格)

### 유형 ① 재성생살(財星生殺)

```
예시: 甲木 일간, 申월(庚金 칠살), 천간에 戊土 편재
      戊土가 庚金을 생(土生金) → 칠살이 더욱 강해짐 → 패격
```

### 유형 ② 제살과다(制殺過多)

제어가 오히려 지나쳐 살이 완전히 무력화될 때도 문제가 된다. 살이 완전히 없어지면 칠살격이 성립되지 않는다.

```
예시: 申월(庚金 칠살), 丙火 식신 여럿 → 살이 완전 제압
      → 칠살격의 살 자체가 없어져 격 소멸
```

### 유형 ③ 식신피합(食神被合)

제살 역할의 식신이 다른 오행과 합거(合去)되어 제살 기능을 잃을 때 패격이 된다.

---

## 6. 구격(救格)

### 유형 ① 제재보살(制財保殺)

재성이 살을 생하여 패격이 될 때, 식상이나 비겁이 재성을 제어하여 살의 과도한 강화를 막는다.

### 유형 ② 합재유식(合財留食)

기신 재성이 합거되어 식신의 제살 기능이 살아날 때 구격이 된다.

### 유형 ③ 거인유식(去印留食)

인성이 식신을 극하여 패격이 될 때, 재성이 인성을 제어(財克印)하여 식신의 제살 기능을 회복시킨다.

---

## 7. 칠살격의 특수 구조

**살인상생(殺印相生)**: 칠살이 인성을 생하고 인성이 일간을 생하는 구조. 무관(武官)의 명이 아닌 특수 권력 구조로 나타난다.

**식신제살(食神制殺)의 귀격**: 식신이 칠살을 완전히 제어하되, 재성이 없어 식신의 제살 기능이 온전할 때 가장 귀하다.

**양인합살(羊刃合殺)**: 일간이 양인(羊刃)을 가지고 칠살과 합하는 경우. 일부 학파에서 무장(武將)의 명으로 해석하나 자평진전은 이 구조를 복잡하게 본다.

---

## 8. 정관격 vs 칠살격 비교

| | 정관격 | 칠살격 |
|---|---|---|
| 극의 성격 | 음양 다름 — 부드러운 제어 | 음양 같음 — 강한 압박 |
| 핵심 용신 | 재성·인성 | 식신·인성 |
| 재성 역할 | 용신(재생관) | 기신(재생살) |
| 식신 역할 | 큰 의미 없음 | 핵심 용신(식신제살) |
| 상관 역할 | 기신(상관견관) | 제살 역할 가능(불안정) |
| 성격 키워드 | 규범, 명예, 문관 | 권력, 압박, 무관 |

---

## 9. 학파 간 이견

**이견 ①** 재성생살의 절대성: 자평진전은 재성이 살을 생하면 반드시 기신이라 하지만, 일부 학파는 신강한 사주에서 재생살도 감당 가능하다고 본다.

**이견 ②** 상관제살의 유효성: 자평진전은 식신제살을 최선으로 보지만, 상관제살도 어느 정도 유효하다는 견해가 있다. 단 불안정함은 공통 인정.

**이견 ③** 칠살독살의 기준: 신강의 기준이 학파마다 달라 칠살독살 인정 여부가 다르다.
