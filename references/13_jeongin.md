# 자평진전(子平眞詮) — 정인격(正印格)

> 자평진전 격국론 시리즈 3편.

---

## 1. 정의와 본질

정인격(正印格)은 **월지 장간 중 정인(正印)이 천간에 투출**하여 성립한다.
정인(正印)이란 일간을 음양이 다르게 생(生)하는 오행이다.

| 일간 | 정인 |
|---|---|
| 甲(양목) | 癸(음수) |
| 乙(음목) | 壬(양수) |
| 丙(양화) | 乙(음목) |
| 丁(음화) | 甲(양목) |
| 戊(양토) | 丁(음화) |
| 己(음토) | 丙(양화) |
| 庚(양금) | 己(음토) |
| 辛(음금) | 戊(양토) |
| 壬(양수) | 辛(음금) |
| 癸(음수) | 庚(양금) |

**정인의 본질**: 어머니가 자식을 낳고 기르는 형상. 일간을 생하여 힘을 주고, 학문·명예·도덕을 상징한다. 관성(官星)이 인성을 생하고 인성이 일간을 생하는 관인상생(官印相生)이 정인격의 가장 귀한 구조다.

---

## 2. 성립 조건

월지 장간 중 정인이 천간에 투출하면 격이 성립한다.

**정인격의 전제**: 정인이 너무 많으면 오히려 일간이 나태해진다. 정인이 하나 있어 일간을 생하는 것이 이상적이다.

---

## 3. 용신·희신·기신

### 용신

**① 관성(官星) — 관인상생(官印相生)**

관성이 인성을 생하고, 인성이 일간을 생하는 흐름. 정인격의 가장 고귀한 구조다.

```
구조: 관성(官) →생→ 정인(印) →생→ 일간(我)
```

**② 관성 없을 때 — 인성 자체의 생조**

관성이 없어도 정인이 월령에 득령하여 충분히 강하면, 일간을 생조하여 학문·명예의 기반이 된다.

### 기신

**① 재성(財星) — 재파인(財破印)**

> "재성이 인성을 극하면 인성이 파괴된다 (財破印格)"

재성은 인성을 직접 극(財克印)하는 오행이다. 정인격에서 재성은 가장 치명적인 기신이다.

```
구조: 재성(財) →극→ 정인(印) → 인성 파괴 → 격 손상
```

**② 식상 과다(食傷過多)**

식상이 많으면 일간의 에너지를 지나치게 설기하여 인성의 생조가 무의미해진다.

---

## 4. 성격(成格)

### 유형 ① 관인상생(官印相生)

```
예시: 壬水 일간, 酉월(辛金 정인 투출), 천간에 己土 정관
      己土가 辛金을 생(土生金) → 辛金이 壬水를 생(金生水)
      → 관인상생 완성
```

이것이 정인격 최고 구조다. 관이 인을 통해 일간에 에너지를 전달하는 흐름이 막힘 없다.

### 유형 ② 인수용겁(印綬用劫)

신약(身弱)한 일간이 인성의 도움만으로 부족할 때, 비겁이 함께 일간을 돕는 구조. 단, 비겁이 재성을 극하여 재성(기신)을 제거하는 이중 역할을 하면 더욱 유효하다.

---

## 5. 패격(敗格)

### 유형 ① 재파인(財破印)

```
예시: 壬水 일간, 酉월(辛金 정인), 천간에 甲木 식신·丙火 편재
      丙火가 辛金을 극(火克金) → 정인 파괴 → 패격
```

### 유형 ② 인다신약(印多身弱) — 역설적 패격

인성이 너무 많으면 오히려 일간이 인성에 의존하여 나태해지고, 인성의 에너지가 일간을 짓누른다. 자평진전은 이 상황도 격의 손상으로 본다.

---

## 6. 구격(救格)

### 유형 ① 겁재제재(劫財制財)

재성이 인성을 극하여 패격이 될 때, 비겁·겁재가 재성을 극하여 인성을 보호한다.

```
예시: 재성(기신)이 있을 때 겁재가 재성을 극(比劫剋財)
      → 인성 보호 → 구격
```

### 유형 ② 관성제겁(官星制劫)

비겁이 너무 많아 일간이 강하고 재성을 탈취할 위험이 있을 때, 관성이 비겁을 제어하여 균형을 잡는다.

---

## 7. 정인격의 특수 구조

**도식격(倒食格)**: 편인(偏印)이 식신을 극하는 구조. 정인격과 달리 편인격에서 주로 나타나지만, 정인이 과다할 때도 식신을 억압하는 유사 현상이 생긴다.

**인수용식(印綬用食)**: 정인이 강하고 관성 없을 때, 식상으로 일간의 에너지를 설기하여 균형. 이 경우 식상이 용신이 되는 이례적 구조.

---

## 8. 학파 간 이견

**이견 ①** 재파인의 절대성: 자평진전은 재성이 있으면 반드시 패격이라 하지만, 일부 학파는 재성이 지지에 있고 천간에 없으면 정인격이 성립 가능하다고 본다.

**이견 ②** 인다신강의 평가: 인성이 많아 신강한 사주를 자평진전은 문제로 보지만, 일부 현대 학파는 신강 자체를 부정적으로만 보지 않는다.

---

## 9. 정인격 핵심 요약

성격: 관인상생(관성이 인성을 생하고 인성이 일간을 생)
패격: 재파인(재성이 인성을 극파)
구격: 비겁이 재성을 제어하여 인성 보호
