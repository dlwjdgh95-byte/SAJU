# 사주 전문가 스킬 (GitHub 실시간 연동판)

자평진전(子平眞詮) 격국론, 궁통보감(窮通寶鑑) 조후론, 적천수(滴天髓) 강약론을 기반으로 한 학문적 사주 해석 스킬이다.

이 파일은 **얇은 로더(loader)**다. 핵심 원리·라우팅·Errata만 인라인으로 담고, 상세 자료(28개 reference + 계산 스크립트)는 **GitHub에서 매번 최신본을 읽어와** 사용한다. 그래서 GitHub 저장소를 업데이트하면 이 스킬의 동작도 자동으로 최신화된다.

## 핵심 원칙 — 사주 해석의 절대 기준 (인라인)

명리학자가 사주를 논리적으로 해독할 때의 절대 기준은 **월령(月令) 기반 격국 파악, 조후(계절 환경) 조절, 기신을 제어해 그릇을 키우는 구격(救格) 원리**다. 이 셋이 맞물려 한 사람의 명(命)을 입체적으로 그린다.

1. **월령·투출·통근**: 월령은 강령("月令乃提綱之最重"). 월지 장간이 천간에 드러나면 투출 → 格이 뚜렷. 천간이 지지에 같은 오행을 두면 통근 → 힘을 얻음. **통근 없는 천간은 뿌리 없는 나무.**
2. **격국의 성패·구응**: 성격 3기준 = ① 격 핵심 오행 월령 득기 ② 용신 건재 ③ 기신 없거나 완전 제어. **패중취귀(敗中取貴)** — 패격이라도 구신이 있으면 귀해진다(예: 상관패인).
3. **조후 역전**: 계절에 따라 같은 오행이 용신↔기신으로 뒤집힌다. **절대적 용신·기신은 없다.** (한여름 丙火에 壬水는 용신, 한겨울 丙火에 壬水는 기신)
4. **학파 이견**: 格을 먼저 보고 억부는 格 내 보조. 격국·조후는 병용(格 성패 먼저, 조후로 보정). 이견 시 단정하지 말고 두 견해를 근거와 함께 제시. → 상세는 `references/23_hakpa_igyeon.md`.

## 분석 절차 (Step 요약 — 상세는 각 파일에서)

**Step 0 — 입력 형태 파악**: (A) 생년월일시 → Step 1 / (B) 사주팔자 직접 입력 → `--pillars`로 십신·장간·통근 산출 후 Step 3부터.

**Step 1 — 사주·대운 계산 (반드시 스크립트)**: GitHub에서 `scripts/saju_calc.py`를 가져와 실행한다. 산문 암산 금지.
```bash
pip install ephem --break-system-packages          # 최초 1회 (절기 계산)
python3 saju_calc.py --birth "1995-10-29 06:30" --gender M
python3 saju_calc.py --pillars "乙亥 丙戌 癸巳 乙卯"
python3 saju_calc.py --year-fortune 2026            # 세운+월 절입표
```
스크립트가 처리: 진태양시 보정(기본 서울 −32분), 절기 기반 월주, 입춘 기준 연주, 자시 규칙, 대운 방향(**연간** 기준 양남음녀)·대운수·간지열, 통근 등급, 월지 장간 투출.
출력 후 확인: `경고` 필드(절입 ±24h·서머타임·1954~61 UTC+8:30), 정각 출생(이전 시지), 음력이면 양력 변환 확인.
> 코드 실행이 불가한 환경이면 `references/26_manual_calc.md`를 읽어 수동 계산한다.

**Step 2 — 기본 정보 해석**: 스크립트의 십신·장간·오행 분포·통근을 그대로 사용. 십신 검산: 나를 生=인성 / 동일 오행=비겁 / 내가 生=식상 / 내가 克=재성 / 나를 克=관성 (음양 같으면 편, 다르면 정).

**Step 3 — 합충 확인**: 천간합(甲己·乙庚·丙辛·丁壬·戊癸), 삼합(寅午戌火·申子辰水·亥卯未木·巳酉丑金), 육합, 지지충. 세부는 `references/22_hapchung_shinsal.md`.

**Step 3-5 — 강약·종격 판단 (격국 전 선행)**: `references/21_jeokcheonsu_gangyak.md`. 통근·월령 득기·천간 생조로 신강/신약. 극약이면 종격 검토. 종격은 전체 5~10% 이하 — 남발 금지.

**Step 4 — 격국 분석**: 格은 반드시 월지(月令)에서 출발. ① 월지 장간 투출 확인(투출 있으면 그 십신으로 格) ② 미투출이면 월지 본기(本氣)로 格 결정(천간에 떠 있는 다른 오행으로 정하면 오류). 격 확정 후 → 해당 格 파일(11~18) + `references/19_sungpae_rules.md`의 판단 프로세스. 사례 대조는 원전 `24_japyeong_cases.md` 우선. 격 확정 직후 병약론 선별 체크(편중·설기태심·재다신약·탐재괴인) → 해당 시 `references/25_byeongyak.md`.

**Step 5 — 조후 분석**: 해당 일간의 궁통보감 파일(01~10)을 읽어 출생 월 용신·희신·기신 확인. 격국 용신과 조후 용신 관계 판단(일치=겸용 가치 2배 / 불일치=격국 우선·조후 보조). 겸용 오행이 대운에서 합거·충파되면 피해 2배.

**Step 6 — 종합 해석 출력**: `references/27_output_formats.md` 템플릿. 필수 섹션: 사주팔자 / 기본 정보 / 십신 배치 / 주요 합충 / 격국 분석 / 조후 분석 / 격국의 고저 / 병약론·억부론 검토 / 종합 해석.

**Step 7 — 대운 해석**: 원국 格·용신·기신 확정 후, 각 대운의 상호작용 판단. 순서 A(대운 십신) → B(용신·기신 관계) → C(합·충) → D(종합, 천간=상반기·지지=하반기). 재다신약·구신 합거·월지 합거·삼합에 묶인 뿌리 등 주의. 格별 용신·기신 빠른표와 상세는 원본/해당 파일 참조. **재격에서 식상은 용신(식상생재)** — 기신 아님.

**Step 8 — 세운 분석**: `--year-fortune`로 간지·절입표 산출. 원국 8 + 대운 2 + 세운 2 = 최대 12자 동시 작용. 우선순위 천간합>지지충>삼합. 충산합해(충이 합을 깸), 세운-대운 관계(가속/역전) 판단. 희신 천간+기신 지지 = 단순 중립 아닌 '기능 혼탁'(예: 壬水+辰土 기토탁임).

**Step 9 — 월운 분석**: 최대 14자 동시 작용. 층위 강도 원국>대운>세운>월운. 월운은 발현 시점(timing) 결정. 세운 기조 안에서 읽는다. 출력은 `references/27_output_formats.md`.

## 주의사항

- **절기 경계일**: 절입 시각 불확실 시 양쪽 가능성 제시.
- **학파 이견**: 격국(자평진전)·강약(적천수)·조후(궁통보감) 충돌 시 두 견해 제시. 상세 `references/23_hakpa_igyeon.md`.
- **종격**: 극소수만 해당, 남발 금지. "신약"과 "종격"은 다른 판단.
